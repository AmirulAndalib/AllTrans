AllTrans 做什么

它将替换在语言中从一种语言到另一种在运行时在应用程序中的所有文本。

说，例如一个应用程序是在德国。用户选择的应用程序名称和所需的语言转换 （说德语到英语）。

然后只要用户使用所需的应用程序，所有的文本，到处都在应用程序中，取而代之的是他们的英语等同。

这有点类似于谷歌翻译中铬的工作的方式。

注意-由于技术的限制，这不会使用很多游戏。没有什么可以呢。

如何使用 AllTrans

此应用程序，需要你报名参加免费从 Microsoft 或 Yandex 翻译的关键。有关如何获取密钥，请参阅下面的说明。

Yandex 翻译支持更多的语言，有多容易的注册过程，并且需要没有信用卡。

微软翻译给更好的翻译，但需要一张信用卡来报名。(但别担心，什么都不会收取.)

  1. 请确保 AllTrans 在筒靴安装程序中启用应用程序-> 模块。
  2. 启动 AllTrans 应用程序。在"全局设置"选项卡上，启用"使用 Yandex 相反的微软"如果你想要使用 Yandex。禁用使用微软。
  3. 选择"从语言翻译"和"翻译成语言"。
  4. 在"翻译应用程序"选项卡中，找到你想要翻译，请单击它旁边的复选框的应用程序。如果该复选框不可见，请尝试旋转横盘整理您的手机变成风景模式。
  5. 关闭并重新启动该应用的程序，你想要翻译 — — 它应该被翻译 ！

如何获得 Yandex 翻译关键

  1. Yandex 翻译 API 网站去
  2. 如果你不已经有一个帐户，请单击"注册"，注册一个新帐户。
  3. 后签约，同意用户协议，然后单击"获取 API 密钥"。
  4. 你已经顺利收到您的密钥 ！

订阅关键是类似于"trnsl.1.1.201701..."

如何获得微软 Azure 翻译关键

  1. 报名参加一个微软 Azure 帐户。
    如果你不已经有 Azure 的帐户，注册一个微软 Azure 帐户。
  2. 您有一个帐户后，登录到Azure 门户。
  3. 添加到 Azure 帐户"微软翻译 API 订阅"。
    1. 选择"现收现付"订阅。将需要一张信用卡，但别担心，没有什么将会收取。
    2. 选择"+ 新"选项。
    3. 从服务列表中选择"智能化"。
    4. 选择"认知服务 Api"。
    5. 选择"API 类型"选项。
    6. 选择"文本翻译"。
    7. 在"定价层"部分中，选择"免费层 (F0)"。
    8. 填写表格的其它部分，然后按"创建"按钮。
  4. 检索你的"身份验证密钥"。
    1. 转到"所有资源"并选择您订阅了微软翻译 API。
    2. 转到"键"选项并复制您订阅的密钥来访问服务。

订阅关键是类似于"321dcba..."

疑难解答

没有应用程序在任何应用程序中正在翻译

  • 如果没有应用程序正被译，检查是在筒靴安装程序中启用"AllTrans"-> 模块。
  • 也请检查您的"微软/Yandex 订阅密钥"是否正确。

翻译的应用被困在打开的屏幕

  • 单击应用程序的名称在"应用程序翻译"选项卡，然后单击"单击到明确翻译缓存"。
  • 如果这是你开始翻译这个应用程序的第一次，单击应用程序的名称在"应用程序翻译"选项卡上，和在"其他设置"填充"延迟之前 Replcaing 与翻译文本中"到 2000 年。

翻译应用程序是"力关闭"

  • 单击应用程序的名称在"应用程序翻译"选项卡，然后单击"单击到明确翻译缓存"。
  • 单击应用程序的名称在"应用程序翻译"选项卡上，然后在"其他设置"填写"延迟之前 Replcaing 与翻译文本"至 2000 年。
  • 单击应用程序的名称在"应用程序翻译"选项卡，然后在"其他设置"中填写"延迟之前开始到翻译 WebViews"到 2000 年。

一个特定的应用程序不是正在翻译

  • 单击应用程序的名称在"应用程序翻译"选项卡，然后在"其他设置"中填写"延迟之前开始到翻译 WebViews"到 2000 年。
  • 如果该应用程序的某些部分仍然不正在翻译，请单击应用程序的名称在"应用程序翻译"选项卡，然后启用"侵略性模式"。

游戏应用程序不是正在翻译

  • 由于技术的限制，"AllTrans"不会与许多游戏。没有什么可以呢。
