Vad AllTrans gör

Det ersätter all text i en app i ett språk från ett språk till ett annat vid körning.

Säger till exempel en app är på tyska. Användaren väljer appens namn och önskat språk omvandlingen (säga tyska till engelska).

Sedan när användaren använder appen krävs, all text, någonstans i appen, ersätts de engelska motsvarigheterna.

Detta är något liknande som Google Translate fungerar i Chrome.

Obs - på grund av en teknisk begränsning, detta fungerar inte med många spel. Ingenting kan göras om detta.

Hur du använder AllTrans

Detta program kräver dig att registrera dig för gratis en nyckel från Microsoft eller Yandex Översätt. Se instruktioner nedan om hur du hämtar nycklarna.

Yandex Översätt stöder flera språk, har en mycket enklare registreringsprocessen och kräver nr kreditkort.

Microsoft Translate ger bättre översättningar, men kräver ett kreditkort för att registrera dig. (Men oroa dig inte, ingenting kommer att laddas.)

  1. Kontrollera AllTrans app är aktiverad i Xposed Installer -> moduler.
  2. Starta appen och AllTrans. ”Globala inställningar” på fliken aktivera 'Använda Yandex istället för Microsoft' om du vill använda Yandex. Inaktivera att använda Microsoft.
  3. Välj ”Översätt från språk” och ”översätta till språket”.
  4. I fliken ”appar att översätta” hitta den app du vill översätta, klicka på kryssrutan bredvid den. Om kryssrutan inte är synlig, försök vrida telefonen i sidled i liggande läge.
  5. Stäng och starta om appen du vill ha översatt - det ska översättas!

Hur till få Yandex översätta nyckel

  1. Gå till Yandex översätta API hemsida
  2. Om du inte redan har ett konto, klicka på ”registrera” och registrera dig för ett nytt konto.
  3. Efter undertecknande upp, godkänner användaravtalet och klicka på ”få API-nyckel”.
  4. Du har framgångsrikt fått din nyckel!

En prenumerationsnyckel är något lik ”trnsl.1.1.201701...”

Hur du får Microsoft Azure översätta nyckel

  1. Registrera dig för en Microsoft Azure-konto.
    Om du inte redan har ett Azure-konto, registrera dig för ett Microsoft Azure-konto.
  2. När du har ett konto, logga in på Azure Portal.
  3. Lägga till en ”Microsoft Translator API prenumeration” i ditt Azure-konto.
    1. Välj en ”pay-as-you-go”-prenumeration. Kräver ett kreditkort, men oroa dig inte, ingenting kommer att debiteras.
    2. Välj alternativet ”+ ny”.
    3. Välj ”intelligens” i listan över tjänster.
    4. Välj ”kognitiva tjänster APIs”.
    5. Välj alternativet ”API-typ”.
    6. Välj ”Textöversättning”.
    7. I avsnittet ”prissättning Tier” välja ”gratis nivå (F0)”.
    8. Fyll i resten av formuläret och tryck på knappen ”Skapa”.
  4. Hämta din ”Authentication Key”.
    1. Gå till ”alla resurser” och välj Microsoft Translator API du prenumererar på.
    2. Gå till alternativet ”nycklar” och kopiera licenskoden för att få tillgång till tjänsten.

En prenumerationsnyckel är något lik ”321dcba...”

Felsökning

Ingen app översätts i någon app

  • Om ingen app översätts, kontrollera är ”AllTrans” är aktiverat i Xposed Installer -> moduler.
  • Kontrollera också om din ”Microsoft/Yandex abonemang nyckel” är korrekt.

Översatt appen har fastnat i startfönstret

  • Klicka på appens namn i ”Apps att översätta” fliken, och klicka på ”Klicka för tydlig översättning Cache”.
  • Om detta är första gången du startade översätta denna app, klicka på appens namn i ”Apps att översätta”-fliken, och i ”övriga inställningar” fylla i ”fördröjning innan Replcaing med översatt Text” till 2000.

Appen översatt är ”Force Close”

  • Klicka på appens namn i ”Apps att översätta” fliken, och klicka på ”Klicka för tydlig översättning Cache”.
  • Klicka på appens namn i ”Apps att översätta”-fliken, och i ”övriga inställningar” fylla i ”fördröjning innan Replcaing med översatt Text” till 2000.
  • Klicka på appens namn i ”Apps att översätta” fliken, och fyll i ”fördröjning innan start att översätta WebView” till 2000 i ”övriga inställningar”.

En viss app översätts inte

  • Klicka på appens namn i ”Apps att översätta” fliken, och fyll i ”fördröjning innan start att översätta WebView” till 2000 i ”övriga inställningar”.
  • Om vissa delar av appen fortfarande inte översätts, klicka på appens namn i ”Apps att översätta” fliken, och aktivera ”aggressivt läge”.

Ett spel app översätts inte

  • På grund av tekniska begränsningar fungerar ”AllTrans” inte med många spel. Ingenting kan göras om detta.

Om du fortfarande har problem som ”Force Close” eller delar av en app som inte översätts

Kontakta mig för stöd. När du gör, kommer att jag behöva iallafall följande information-

  1. Namnet på appen, tillsammans med en länk att ladda ner appen från vissa APK store som Google Play Store, CoolApk, etc.
  2. Din telefons Android-version.
  3. En ”logcat” av försöker du öppna appen som ger dig problem. Du kan använda fritt tillgängliga appar i Play butik för detta, som denna

När du har ovanstående, kontakta mig på den XDA stöd tråd.
Du kan också rapportera problem på den projektets Github.
