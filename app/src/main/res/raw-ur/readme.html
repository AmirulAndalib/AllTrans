کیا اللٹرنس کرتا ہے

اس میں ایک ایک زبان سے زبان دوسرے رن ٹائم پر تمام متن ایک اپلی کیشن میں بدل دیتا ہے ۔

مثال کے طور ایک اپلی کیشن میں جرمن کہتے ہیں ۔ ایک صارف نام اپلی کیشن اور مطلوبہ زبان بدلی (کہتے ہیں کہ جرمن سے انگریزی) منتخب کرتا ہے ۔

پھر جب صارف مطلوب اپلی کیشن کا استعمال کرتا ہے، تمام، اپلی کیشن, میں انیوہیری ہیں بدلا گیا متن ان کےانگریزی کی طرف سے ۔

یہ کچھ اسی طرح جس طرح گوگل ترجمہ Chrome میں کام کرنا ہے ۔

نوٹ - ایک تکنیکی حدود کی وجہ سے یہ بہت سے کھیل کے ساتھ کام نہیں کرے گا ۔ کچھ بھی نہیں اس کے بارے میں کیا جا سکتا ہے ۔

اللٹرنس کا استعمال کیسے کریں

اس ایپلی کیشن کی ضرورت ہے آپ مفت کے لئے ایک چابی کے Microsoft یا yandex کی ترجمہ سے سائن اپ کریں ۔ ذیل کی ہدایات کی چابیاں حاصل کرنے کے لئے کس طرح پر دیکھیں ۔

Yandex کی ترجمہ زیادہ زبانوں کی حمایت کرتا ہے، ایک بہت آسان سائن اپ پراسس ہے، اور کوئی کریڈٹ کارڈکی ضرورت ہے ۔

Microsoft ترجمہ بہتر ترجمہکرتا ہے، لیکن سائن اپ کے لیے ایک کریڈٹ کارڈ کی ضرورت ہے ۔ (مگر پریشان مت ہوں، کچھ نہیں ہوگا چارج.)

  1. اس بات کا یقین کر لیں کہ ماڈیول اپلی کیشن اپوساد انسٹالر میں فعال ہے اللٹرنس ->.
  2. اللٹرنس ایپ کا آغاز ہے ۔ آپ yandex کی استعمال کرنا چاہتے ہیں تو "عالمی ترتیبات" کے ٹیب میں, "استعمال Yandex کی بجائے Microsoft" فعال کریں ۔ Microsoft استعمال کرنے کے لیے نااہل ہے ۔
  3. وہ "زبان سے ترجمہ" اور "زبان کا ترجمہ" منتخب کریں ۔
  4. "اطلاقات کے لئے ترجمہ" ٹیب میں، ایپ آپ کا ترجمہ، سے اگلا پڑتال خانہ پر کلک کرنا چاہتے تلاش کریں ۔ پڑتال خانہ مرئی نہیں ہیں تو sideways باری ہے آپ کے فون کو افقی انداز موڈ میں گھومنے کی کوشش کریں ۔
  5. بند کریں اور ترجمہ حسب اپلی کیشن پھر شروع کریں - یہ ترجمہ ہونا چاہیے!

Yandex کی ترجمہ چابی حاصل کرنے کے لئے کس طرح

  1. Yandex کی ترجمہ API کی ویب سائٹ کے لیے جانا
  2. اگر آپ پہلے ہی ایک اکاؤنٹ نہیں ہے تو "رجسٹر" پر کلک کریں اور ایک نیا اکاؤنٹ کے لئے سائن اپ کریں ۔
  3. سائن اپ کے بعد صارف سے متعلق معاہدے پر اتفاق کرتے ہیں اور "API کلید حاصل کریں" پر کلک کریں ۔
  4. آپ نے کامیابی سے آپ کی چابی وصول کی ہے!

رکنیت کی کلید "trnsl.1.1.201701..." کی طرح کچھ ہے ۔

Microsoft Azure ترجمہ چابی حاصل کرنے کے لئے کس طرح

  1. ایک Microsoft Azure اکاؤنٹ کے لئے سائن اپ کریں ۔
    آپ پہلے ہی ایک Azure اکاؤنٹ نہیں ہے تو، ایک Microsoft Azure اکاؤنٹکے لئے سائن اپ کریں ۔
  2. اس کے بعد کہ آپ کا اکاؤنٹ ہے، Azure پورٹلمیں سائن ان کریں ۔
  3. "Microsoft مترجم API رکنیت" اپنے Azure اکاؤنٹ شامل کریں ۔
    1. ایک "pay-as-you-go" کی رکنیت کا انتخاب کریں ۔ کریڈٹ کارڈ کی ضرورت ہوگی، لیکن فکر نہ کرو، کچھ بھی نہیں چارج کیا جائے گا ۔
    2. "+ نئی" وہ اختیار منتخب کریں ۔
    3. "انٹیلی جنس" خدمات کی فہرست سے منتخب کریں ۔
    4. "ادراک خدمات APIs" منتخب کریں ۔
    5. "API قسم" وہ اختیار منتخب کریں ۔
    6. "متن ترجمہ" منتخب کریں ۔
    7. "مفت درجے (F0)'"درجے کی قیمتوں کا تعین"کے سیکشن میں، منتخب کریں ۔
    8. باقی فارم کے باہر بھرنے اور "بنائیں" بٹن دبائیں ۔
  4. آپ "تصدیق کی کلید" بازیافت کریں ۔
    1. 'کے لئے تمام وسائل' جائیں اور آپ اپنایا Microsoft مترجم API منتخب کریں ۔
    2. "چابیاں" اختیار کے لیے جائیں اور سروس تک رسائی کے لیے آپ رکنیت کی چابی نقل کریں ۔

کوئی اپلی کیشن کوئی اپلی کیشن میں ترجمہ کیا جارہا ہو

  • اگر کوئی اپلی کیشن میں ترجمہ کیا جارہا ہو تو چیک ان "اللٹرنس" ہے اپوساد انسٹالر میں ہے اہل ماڈیول -> ۔
  • بھی چیک کریں اگر آپ "Microsoft/yandex کی رکنیت کلید" صحیح ہے ۔

ترجمہ اپلی کیشن افتتاحی سکرین میں پھنس جاتا ہے

  • اپلی کیشن کے نام میں "اطلاقات کا ترجمہ کرنے کے لئے" پر کلک کریں tab منتخب کریں، اور "پر کلک کریں کے لئے واضح ترجمہ کیسہ پر" پر کلک کریں ۔
  • اگر اس نے پہلی بار آپ نے اس اپلی کیشن کا ترجمہ شروع کر دیا ہے، اپلی کیشن کے نام جدول "اطلاقات کے لئے ترجمہ" اور "تاخیر سے پہلے ریپلکینگ متن کے ساتھ ترجمہ" میں "دیگر ترتیبات" بھریں میں 2000 سے پر کلک کریں ۔

ٹرانسلیٹد ایپ "قریبی فورس" ہے

  • اپلی کیشن کے نام میں "اطلاقات کا ترجمہ کرنے کے لئے" پر کلک کریں tab منتخب کریں، اور "پر کلک کریں کے لئے واضح ترجمہ کیسہ پر" پر کلک کریں ۔
  • اپلی کیشن کے نام "اطلاقات کے لئے ترجمہ" کے ٹیب پر کلک کریں اور میں "دیگر ترتیبات" "تاخیر سے پہلے ریپلکینگ کے ساتھ ترجمہ متن میں" 2000 سے بھریں ۔
  • اپلی کیشن کے نام میں "اطلاقات کا ترجمہ کرنے کے لئے" پر کلک کریں tab منتخب کریں، اور "دیگر ترتیبات میں" پُر "تاخیر سے پہلے شروع کرنے کے لیے ترجمہ ویبویوس میں" 2000 کے لیے ۔

ایک مخصوص اپلی کیشن کا ترجمہ نہیں کیا جا رہا ہے

  • اپلی کیشن کے نام میں "اطلاقات کا ترجمہ کرنے کے لئے" پر کلک کریں tab منتخب کریں، اور "دیگر ترتیبات میں" پُر "تاخیر سے پہلے شروع کرنے کے لیے ترجمہ ویبویوس میں" 2000 کے لیے ۔
  • اپلی کیشن کے کچھ حصے اب بھی ترجمہ کیا جارہا ہیں نہیں تو "اطلاقات کا ترجمہ کرنے کے لئے" میں اپلی کیشن کے نام پر کلک tab منتخب کریں، اور "جارحانہ موڈ" اہل بنائیں ۔

ایک کھیل ایپ کا ترجمہ نہیں کیا جا رہا ہے

  • کی وجہ سے تکنیکی حدود، "اللٹرنس" بہت سے کھیل کے ساتھ کام نہیں کرے گا ۔ کچھ بھی نہیں اس کے بارے میں کیا جا سکتا ہے ۔
