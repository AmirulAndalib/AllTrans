O que faz AllTrans

Substitui todo o texto em um app em um idioma de um idioma para outro em tempo de execução.

Dizer por exemplo que um app é em alemão. Um usuário seleciona o nome do app e a conversão de idioma desejado (dizer em alemão para o inglês).

Então sempre que o usuário utiliza o aplicativo necessário, todo o texto, em qualquer lugar no aplicativo, são substituídos por seus equivalentes em inglês.

Isto é algo semelhante à maneira como o Google Translate funciona no Chrome.

Nota - devido a uma limitação técnica, isso não vai funcionar com muitos jogos. Nada pode ser feito sobre isso.

Como usar AllTrans

Este aplicativo requer que você se inscrever para free uma chave da Microsoft ou traduza Yandex. Consulte as instruções abaixo sobre como obter as chaves.

Yandex Translate suporta mais idiomas, tem um processo de inscrição muito mais fácil e requer sem cartão de crédito.

Microsoft Translate dá melhores traduções, mas requer um cartão de crédito para se inscrever. (Mas não se preocupe, nada será cobrado.)

  1. Certifique-se de AllTrans app está habilitado no instalador Xposed -> módulos.
  2. AllTrans app de lançamento. Na guia "Configurações globais", habilite "Usar Yandex instead of Microsoft" se você quiser usar o Yandex. Desabilitar para usar o Microsoft.
  3. Escolha a "Tradução do idioma" e "Traduzir para a língua".
  4. Na aba "Apps para traduzir", encontre o app que você deseja traduzir, clique a caixa de seleção ao lado dele. Se a caixa de seleção não estiver visível, tente girar seu telefone lateralmente em modo paisagem.
  5. Feche e reinicie o aplicativo que você quer traduzida - que deve ser traduzido!

Como obter a chave de traduzir Yandex

  1. Ir a Yandex traduzir site API
  2. Se você não tiver uma conta, clique em "registrar" e Cadastre-se para uma nova conta.
  3. Após a inscrição, concordo com o contrato de usuário e clique em "Get API Key".
  4. Você recebeu sua chave com sucesso!

A chave de assinatura é algo como "trnsl.1.1.201701..."

Como obter o Microsoft Azure definição da palavra chave

  1. Cadastre-se para uma conta do Microsoft Azure.
    Se você não tiver uma conta Azure, Cadastre-se para uma conta do Microsoft Azure.
  2. Depois que você tiver uma conta, Cadastre-se no Portal do Azure.
  3. Adicione uma "API Microsoft Translator"assinatura para sua conta Azure.
    1. Escolha uma assinatura "pré-pago". Exigirá um cartão de crédito, mas não se preocupe, nada será cobrado.
    2. Selecione a opção "+ nova".
    3. Selecione "Inteligência" da lista de serviços.
    4. Selecione "APIs de serviços cognitivo".
    5. Selecione a opção "Tipo de API".
    6. Selecione "Tradução de texto".
    7. Na seção de "Camada de preços", selecione a "camada de enciclopédia (F0)".
    8. Preencha o resto do formulário e pressione o botão "Criar".
  4. Recupere sua "chave de autenticação".
    1. Ir a "Todos os recursos" e selecione a API do Microsoft Translator você inscrito.
    2. Vá até a opção de "Chaves" e copiar a sua chave de assinatura para acessar o serviço.

A chave de assinatura é algo como "321dcba..."

Solução de problemas

Nenhum app está sendo traduzido em qualquer app

  • Se nenhum aplicativo está a ser traduzido, o cheque é "AllTrans" está habilitado no instalador Xposed -> módulos.
  • Verifique também se a sua "chave de assinatura do Microsoft/Yandex" está correto.

Traduzido do app está preso na tela de abertura

  • Clique no nome do aplicativo em "Apps para traduzir" guia e clique em "Clique para limpar tradução Cache".
  • Se esta é a primeira vez que você começou a traduzir este aplicativo, clique no nome do aplicativo na aba "Apps para traduzir" e em "Outras configurações" preencher "Atraso antes de Replcaing com traduzido texto" de 2000.

O app traduzido é "Força perto"

  • Clique no nome do aplicativo em "Apps para traduzir" guia e clique em "Clique para limpar tradução Cache".
  • Clique no nome do aplicativo na aba "Apps para traduzir" e em "outras configurações" preencher "Atraso antes de Replcaing com traduzido texto" de 2000.
  • Clique no nome do aplicativo em "Apps para traduzir" guia e em "Outras configurações" preencher "Atraso antes de começar a traduzir WebViews" de 2000.

Uma aplicação em particular não está a ser traduzida

  • Clique no nome do aplicativo em "Apps para traduzir" guia e em "Outras configurações" preencher "Atraso antes de começar a traduzir WebViews" de 2000.
  • Se algumas partes do app não ainda estão sendo traduzidos, clique no nome do aplicativo em "Apps para traduzir" tab e ativar o "Modo agressivo".

Um app jogo não está a ser traduzido

  • Devido a limitações técnicas, "AllTrans" não vai funcionar com muitos jogos. Nada pode ser feito sobre isso.
