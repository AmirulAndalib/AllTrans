Apakah AllTrans

Ia menggantikan semua teks dalam aplikasi dalam bahasa dari satu bahasa lain di runtime.

Katakan sebagai contoh aplikasi adalah dalam Bahasa Jerman. Pengguna memilih nama aplikasi, dan penukaran bahasa yang dikehendaki (katakan Jerman kepada Inggeris).

Kemudian apabila pengguna menggunakan aplikasi diperlukan, Semua teks, ANYWHERE dalam aplikasi, digantikan oleh Bahasa Inggeris mereka setara.

Ini adalah sesuatu yang serupa dengan cara bekerja Terjemahan Google Chrome.

Nota - kerana had teknikal, ini tidak akan bekerja dengan permainan banyak. Apa-apa yang boleh dibuat tentang perkara ini.

Bagaimana untuk menggunakan AllTrans

Permohonan ini memerlukan anda untuk mendaftar untuk percuma kekunci dari Microsoft atau terjemahan Yandex. Sila lihat arahan di bawah tentang bagaimana untuk mendapatkan kunci.

Terjemahan Yandex menyokong lebih banyak bahasa, mempunyai proses yang lebih mudah pendaftaran, dan memerlukan NO kad kredit.

Microsoft Translate memberikan terjemahan yang lebih baik, tetapi memerlukan kad kredit untuk mendaftar. (Tetapi jangan risau, tiada apa-apa akan dikenakan.)

  1. Pastikan AllTrans aplikasi didayakan dalam pemasang Xposed -> modul.
  2. Melancarkan aplikasi AllTrans. Dalam tab "Global tetapan", membolehkan "Menggunakan Yandex instead of Microsoft" Jika anda ingin menggunakan Yandex. Melumpuhkan untuk menggunakan Microsoft.
  3. Pilih "Translate dari Bahasa" dan "Terjemah ke bahasa".
  4. Di tab "Apl untuk menterjemah", menemui aplikasi yang anda mahu untuk menterjemah, klik kotak semak bersebelahan dengannya. Jika kotak semak segerak yang tidak boleh dilihat, cuba berputar telefon anda ke arah sisi ke mod landskap.
  5. Tutup dan mulakan semula aplikasi yang hendak diterjemahkan - ia perlu diterjemah!

Bagaimana untuk mendapatkan kunci menterjemah Yandex

  1. Pergi ke Yandex Terjemah laman web API
  2. Jika anda tidak mempunyai akaun, klik "Daftar" dan mendaftar untuk akaun baru.
  3. Selepas mendaftar, bersetuju dengan perjanjian pengguna dan klik "Dapatkan kunci API".
  4. Anda telah berjaya menerima kunci!

Kekunci langganan adalah sesuatu seperti "trnsl.1.1.201701..."

Bagaimana untuk mendapatkan kunci menterjemah Azure Microsoft

  1. Mendaftar untuk akaun Microsoft Azure.
    Jika anda tidak mempunyai akaun biru, mendaftar untuk akaun Microsoft Azure.
  2. Selepas anda mempunyai akaun untuk log masuk ke Azure Portal.
  3. Tambah "Langganan API Microsoft Translator" akaun Hotel.
    1. Memilih langganan "pay-as-you-go". Memerlukan kad kredit, tetapi jangan risau, tiada apa-apa akan dikenakan.
    2. Pilih opsyen "+ baru".
    3. Pilih "Perisikan" daripada Senarai Perkhidmatan.
    4. Pilih "Perkhidmatan kognitif api".
    5. Pilih opsyen "API jenis".
    6. Pilih "Penterjemahan teks".
    7. Dalam bahagian "Harga Tier", pilih yang "percuma Tier (F0)".
    8. Isi borang yang lain, dan tekan butang "Cipta".
  4. Dapatkan "Kunci pengesahan".
    1. Pergi ke "Semua sumber" dan pilih Microsoft Translator API anda melanggan.
    2. Pergi ke opsyen "Kunci" dan salinan kunci langganan untuk mengakses perkhidmatan tersebut.

Kekunci langganan adalah sesuatu seperti "321dcba..."

Penyelesaian masalah

Tiada aplikasi kini sedang diterjemah dalam mana-mana aplikasi

  • Jika tiada aplikasi kini sedang diterjemah, semak adalah "AllTrans" telah dibolehkan dalam pemasang Xposed -> modul.
  • Juga menyemak jika anda "Microsoft/Yandex langganan kunci" adalah betul.

Aplikasi diterjemahkan terperangkap dalam skrin pembukaan

  • Klik pada nama apl dalam "Apl untuk menterjemah" tab dan klik pada "klik untuk jelas terjemahan"tempat persembunyian.
  • Jika ini adalah kali pertama anda mula menterjemah aplikasi ini, klik pada nama apl dalam tab "Apl untuk menterjemah", dan "Lain-lain tetapan" isi "Kelewatan sebelum Replcaing dengan diterjemahkan teks" hingga 2000.

Aplikasi Translated adalah "Tutup tentera"

  • Klik pada nama apl dalam "Apl untuk menterjemah" tab dan klik pada "klik untuk jelas terjemahan"tempat persembunyian.
  • Klik pada nama apl dalam tab "Apl untuk menterjemah" dan dalam "tetapan lain" isi "Kelewatan sebelum Replcaing dengan diterjemahkan teks" hingga 2000.
  • Klik pada nama apl dalam "Apl untuk menterjemah" tab dan di "Lain-lain tetapan" isi "Kelewatan sebelum bermula untuk menterjemah WebViews" hingga 2000.

Aplikasi tertentu kini sedang diterjemah

  • Klik pada nama apl dalam "Apl untuk menterjemah" tab dan di "Lain-lain tetapan" isi "Kelewatan sebelum bermula untuk menterjemah WebViews" hingga 2000.
  • Jika beberapa bahagian apl tersebut masih tidak menjadi diterjemahkan, klik pada nama apl dalam "Apl untuk menterjemah" tab, dan membolehkan "Agresif Mode".

Aplikasi permainan kini sedang diterjemah

  • Disebabkan oleh had teknikal, "AllTrans" tidak akan berfungsi dengan permainan banyak. Apa-apa yang boleh dibuat tentang perkara ini.

Jika anda masih mempunyai masalah seperti "Tutup Force" atau bahagian-bahagian aplikasi tidak telah diterjemahkan

  2. Versi Android telefon bimbit anda.
  3. "Logcat" yang cuba membuka aplikasi yang memberikan masalah. Anda boleh menggunakan aplikasi secara percuma di kedai mainan ini, seperti ini
