En quoi consiste AllTrans

Il remplace tout le texte dans une application dans un langage d’une langue à l’autre lors de l’exécution.

Dire par exemple, une application est en allemand. L’utilisateur sélectionne le nom de l’application et la conversion de la langue souhaitée (dire allemand vers l’anglais).

Ensuite, chaque fois que l’utilisateur utilise l’application requise, tout le texte, n’importe où dans l’application, sont remplacés par leurs équivalents anglais.

C’est quelque chose de semblable à la façon dont Google Translate fonctionne en Chrome.

Note - en raison d’une limitation technique, cela ne fonctionnera pas avec de nombreux jeux. Rien ne peut être fait à ce sujet.

Comment utiliser AllTrans

Cette application nécessite vous inscrire gratuitement à une clé de Microsoft ou Traduisez Yandex. Voir les instructions ci-dessous sur la façon d’obtenir les clés.

Yandex Traduisez prend en charge plusieurs langues, possède un processus beaucoup plus facile de s’inscrire et nécessite aucune carte de crédit.

Translate Microsoft donne des traductions de meilleure qualité, mais nécessite une carte de crédit pour vous inscrire. (Mais ne vous inquiétez pas, rien ne sera facturé.)

  1. Assurez-vous que AllTrans app est activé dans l’installateur Xposed -> Modules.
  2. Lancement d’AllTrans app. Dans l’onglet « Paramètres globaux », activez « Utilisent Yandex instead of Microsoft » si vous voulez utiliser Yandex. Désactiver pour utiliser Microsoft.
  3. Choisissez la « Traduction du langage » et « Traduire à la langue ».
  4. Dans l’onglet « Apps à traduire », recherchez l’application que vous souhaitez traduire, cliquez sur la case à cocher à côté de lui. Si la case à cocher n’est pas visible, essayez de tourner sur le côté, votre téléphone en mode paysage.
  5. Fermez et redémarrez l’application souhaitée traduit - il devrait être traduit !

Comment obtenir la clé de traduire de Yandex

  1. Aller Yandex traduction site Web API
  2. Si vous n’avez pas déjà un compte, cliquez sur « s’inscrire » et inscrivez-vous pour un nouveau compte.
  3. Après la signature, acceptez le contrat d’utilisateur, puis cliquez sur « Get API Key ».
  4. Vous avez reçu avec succès votre clé !

La clé d’abonnement est quelque chose comme « trnsl.1.1.201701... »

Comment obtenir la clé de traduction Microsoft Azure

  1. Inscrivez-vous pour un compte Microsoft Azure.
    Si vous n’avez déjà un compte Azure, inscrivez-vous pour un compte Microsoft Azure.
  2. Après que vous avez un compte, ouvrez une session dans le Portail Azure.
  3. Ajouter un abonnement « Microsoft Translator API » à votre compte Azure.
    1. Choisissez un abonnement « par répartition ». Il faudra une carte de crédit, mais ne vous inquiétez pas, rien ne sera facturé.
    2. Sélectionnez l’option « + nouveau ».
    3. Sélectionnez « Intelligence » dans la liste des services.
    4. Sélectionnez « Services cognitifs APIs ».
    5. Sélectionnez l’option « Type d’API ».
    6. Sélectionnez « Texte traduction ».
    7. Dans la section « Niveau de tarification », sélectionnez le « niveau gratuit (F0) ».
    8. Remplir le reste du formulaire et appuyez sur le bouton « Créer ».
  4. Récupérez votre « clé d’authentification ».
    1. Aller à « Toutes les ressources », puis sélectionnez Microsoft Translator API vous souscrit un abonnement.
    2. Allez à l’option « Clefs » et copier votre clé d’abonnement pour accéder au service.

La clé d’abonnement est quelque chose comme « 321dcba... »

Dépannage

Aucune application n’est en cours de traduction dans n’importe quelle application

  • Si aucune application n’est en cours de traduction, check est « AllTrans » est activé dans l’installateur Xposed -> Modules.
  • Vérifiez également si votre « clé d’abonnement Microsoft/Yandex » est correct.

Traduit app est coincé dans l’écran d’ouverture

  • Cliquez sur le nom de l’application dans « Apps à traduire » onglet, puis cliquez sur « Cliquez pour effacer Cache traduction ».
  • Si c’est la première fois que vous avez commencé à traduire cette app, cliquez sur le nom de l’application dans l’onglet « Apps pour traduire » et dans « Autres paramètres » remplir « Délai avant Replcaing avec traduit texte » 2000.

Le soft de traduction est « Force Close »

  • Cliquez sur le nom de l’application dans « Apps à traduire » onglet, puis cliquez sur « Cliquez pour effacer Cache traduction ».
  • Cliquez sur le nom de l’application dans l’onglet « Apps pour traduire » et en « autres paramètres » remplir « Délai avant Replcaing avec texte cible » pour 2000.
  • Cliquez sur le nom de l’application dans « Apps à traduire » onglet, puis dans « Autres paramètres » remplir « Retard avant de commencer à traduire WebViews » pour 2000.

Une application particulière n'est pas en cours de traduction

  • Cliquez sur le nom de l’application dans « Apps à traduire » onglet, puis dans « Autres paramètres » remplir « Retard avant de commencer à traduire WebViews » pour 2000.
  • Si certaines parties de l’application ne sont toujours pas traduits, cliquez sur le nom de l’application dans « Apps à traduire » onglet, puis activez le Mode « agressif ».

Un jeu app n'est pas en cours de traduction

  • En raison de limitations techniques, « AllTrans » ne fonctionnera pas avec de nombreux jeux. Rien ne peut être fait à ce sujet.
