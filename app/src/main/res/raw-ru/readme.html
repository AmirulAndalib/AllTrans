Что делает AllTrans

Он заменяет весь текст в приложении на языке с одного языка на другой во время выполнения.

Скажем, например приложение находится на немецком языке. Пользователь выбирает имя приложения и требуемого языка преобразования (скажем, немецкий на английский).

Затем каждый раз, когда пользователь использует необходимые приложения, весь текст, любом в приложении, заменяются их английские эквиваленты.

Это нечто похожее на то, что Google Translate работает в Chrome.

Примечание - вследствие технических ограничений, это не будет работать со многими играми. Ничто не может быть сделано по этому поводу.

Как пользоваться AllTrans

Это приложение требует вам подписаться на бесплатный ключ от Microsoft или Yandex перевести. Смотрите инструкции ниже о том, как получить ключи.

Яндекс перевести поддерживает несколько языков, имеет гораздо проще процесс signup и требует нет кредитной карты.

Перевести Microsoft дает лучший перевод, но требует кредитной карты, чтобы зарегистрироваться. (Но не волнуйтесь, ничего не будет взимается.)

  1. Убедитесь, что приложение включен в инсталлятор Xposed AllTrans -> модули.
  2. Запустите приложение AllTrans. В «Глобальные параметры» на вкладке включите «Использовать Microsoft вместо Яндекс», если вы хотите использовать Яндекс. Отключение использования Microsoft.
  3. Выберите «Перевод с языка» и «Перевести на язык».
  4. На вкладке «Apps перевести» найти приложение, которое вы хотите перевести, установите флажок рядом с ним. Если этот флажок не отображается, попробуйте повернуть боком ваш телефон в ландшафтный режим.
  5. Закройте и перезапустите приложение, которое вы хотите перевод - оно должно быть переведено!

Как получить ключ перевод Яндекс

  1. Перейти на Яндекс перевод веб-сайта API
  2. Если вам еще нет учетной записи, нажмите кнопку «Регистрация» и подпишитесь на новую учетную запись.
  3. После подписания, соглашаетесь с пользовательским соглашением и нажмите кнопку «Получить ключ API».
  4. Вы успешно получили ключ!

Ключ подписки что-то вроде «trnsl.1.1.201701...»

Как получить ключ перевести Microsoft Azure

  1. Подпишитесь на учетную запись Microsoft Azure.
    Если вам еще нет учетной записи Azure, подпишитесь на учетную запись Microsoft Azure.
  2. После того, как у вас есть счет, войдите на Портал Azure.
  3. Добавление учетной записи Azure «Microsoft Translator API подписка».
    1. Выберите подписку на «распределительной». Потребует кредитной карты, но не волнуйтесь, ничего не взимается.
    2. Выберите параметр «+ новые».
    3. Выберите в списке услуг «Интеллект».
    4. Выберите «Когнитивная услуг API».
    5. Выберите вариант «API типа».
    6. Выберите «Перевод текстов».
    7. В разделе «Цены уровня» выберите «Свободный уровень (F0)».
    8. Заполните оставшуюся часть формы и нажмите кнопку «Создать».
  4. Получить «Ключ проверки подлинности».
    1. Перейдите к «Все ресурсы» и выберите Microsoft Translator API, вы подписались.
    2. Перейдите к параметру «Ключи» и скопируйте ключ подписки для доступа к службе.

Ключ подписки что-то вроде «321dcba...»

Устранение неполадок

Нет app в настоящее время переводится в любом приложении

  • Если нет app в настоящее время переводится, проверить это «AllTrans» включен в инсталлятор Xposed -> модули.
  • Также проверьте, если ваш «ключ подписки Microsoft/Яндекс» является правильным.

Перевод приложения застрял в открытие экрана

  • Нажмите на название приложения в «Apps перевести» вкладку и нажмите на «Нажмите чтобы очистить кэш перевод».
  • Если это первый раз, вы начали переводить этот app, нажмите на название приложения в вкладку «Apps перевести» и «Прочие настройки» заполните «Задержка перед Replcaing с перевод текста» в 2000.

App перевод близка «силы»

  • Нажмите на название приложения в «Apps перевести» вкладку и нажмите на «Нажмите чтобы очистить кэш перевод».
  • Нажмите на название приложения в «Перевести приложения» на вкладке, и в «прочие настройки» заполнить «Задержка перед Replcaing с перевод текста» в 2000 году.
  • Нажмите на название приложения в «Apps перевести» вкладку и в «Прочие настройки» заполните «Задержка перед начиная чтобы перевести WebViews» в 2000 году.

Особое приложение не переводится

  • Нажмите на название приложения в «Apps перевести» вкладку и в «Прочие настройки» заполните «Задержка перед начиная чтобы перевести WebViews» в 2000 году.
  • Если некоторые части приложения до сих пор не переводятся, нажмите на название приложения в «Apps перевести» вкладку и включить «Агрессивный режим».

Игра app не переводится

  • Из-за технических ограничений «AllTrans» не будет работать со многими играми. Ничто не может быть сделано по этому поводу.
