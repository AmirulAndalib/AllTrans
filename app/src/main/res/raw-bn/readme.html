কি AllTrans

এক ভাষা থেকে আরেক চলার সময় ভাষায় অ্যাপটি সব লেখা প্রতিস্থাপন করে।

যেমন একটি অ্যাপ। জার্মানি বলে। ব্যবহারকারী অ্যাপটি নাম এবং প্রয়োজনীয় ভাষায় রূপান্তর (বলে জার্মান থেকে ইংরেজী) নির্বাচন করে।

এরপর ব্যবহারকারীর প্রয়োজনীয় অ্যাপ ব্যবহার করেন তখন সব, অ্যাপ্লিকেশন, ANYWHERE হয় বদলানো দ্বারা তাদের ইংরেজি প্রতিশব্দ।

যেভাবে গুগল ক্রোম এ কাজ করার মতো কিছু একটা এটা।

নোট - প্রযুক্তিগত সীমাবদ্ধতা কারণে এটি অনেক গেমস নিয়ে কাজ করবে না। তো এই সব করা যেতে পারে।

AllTrans ব্যবহার করতে হয়

এই অ্যাপ্লিকেশন প্রয়োজন আপনি বিনামূল্যে জন্য কী Microsoft বা ইয়ানডেক্স অনুবাদ থেকে সাইন আপ করতে। কিভাবে কী পাওয়ার জন্য নিচের নির্দেশনা দেখুন।

ইয়ানডেক্স অনুবাদ আরো ভাষা সমর্থন করে, অনেক সহজ সাইনআপ পদ্ধতি আছে এবং কোন ক্রেডিট কার্ডেরপ্রয়োজন হয়।

Microsoft অনুবাদ ভালো অনুবাদকরেন, কিন্তু ক্রেডিট কার্ড সাইন আপ প্রয়োজন হয়। (কিন্তু চিন্তা কোরো না, কিছুই হবে চার্জ করা।)

  1. নিশ্চিত করুন যে মডিউল -> AllTrans অ্যাপ Xposed ইনস্টলার মধ্যে সক্রিয় করা হয়।
  2. AllTrans অ্যাপ্লিকেশন কোড চালু করুন। ইয়ানডেক্স ব্যবহার করতে চাইলে "গ্লোবাল সেটিংস" ট্যাবে, "ইয়ানডেক্স বদলে মাইক্রোসফট' ব্যবহার করে সক্রিয় করুন। Microsoft ব্যবহার করতে নিষ্ক্রিয় করে দেয়।
  3. ' ভাষা থেকে অনুবাদ"এবং"ভাষা থেকে অনুবাদ"বেছে নিন।
  4. "অ্যাপ থেকে অনুবাদ" ট্যাবে, অনুবাদ, এর পাশের পরীক্ষণ বাক্সটি টিপুন করতে চান অ্যাপটি পেতে। আপনার ফোনের বাইরে প্রাকৃতিক পরিমণ্ডলে আবর্তিত বাক্স দেখা যায় না, তবে চেষ্টা করুন।
  5. বন্ধ করুন এবং আপনি অনুবাদ অ্যাপ্লিকেশন পুনরায় শুরু - এর অনুবাদ করা উচিত।

কিভাবে ইয়ানডেক্স অনুবাদ কী পাওয়া যাবে

  1. ইয়ানডেক্স API ওয়েবসাইট অনুবাদ করতে করতে যেতে
  2. যদি আপনি ইতিমধ্যেই কোনো অ্যাকাউন্ট নেই, "রেজিস্টার করুন" ক্লিক করুন এবং একটি নতুন অ্যাকাউন্টের জন্য সাইন আপ করুন।
  3. সাইন আপ করা, ব্যবহারকারী চুক্তি করতে সম্মত হন এবং "API কী পেতে" ক্লিক করুন।
  4. আপনি সফলভাবে আপনার কী পেয়েছেন!

সাবস্ক্রিপশন কী '... trnsl.1.1.201701'-এর মতো কিছু একটা।

কীভাবে Microsoft Azure কী অনুবাদ

  1. Microsoft-এর Azure অ্যাকাউন্ট সাইন আপ করুন।
    যদি আপনি ইতিমধ্যে নীল অ্যাকাউন্ট নেই, Microsoft-এর Azure অ্যাকাউন্টসাইন আপ করুন।
  2. Azure পোর্টালএ আপনার একটি একাউন্ট আছে, পর সাইন করুন।
  3. "Microsoft অনুবাদক API সাবস্ক্রিপশন' নীল অ্যাকাউন্টের সাথে যুক্ত করুন।
    1. "Pay-as-you-go" সদস্যতা নির্বাচন করুন। ক্রেডিট কার্ডের প্রয়োজন হবে, কিন্তু চিন্তা কোরো না, কোন চার্জ প্রযোজ্য হবে।
    2. '+ নতুন' অপশন বেছে নিন।
    3. 'গোয়েন্দা' সেবা তালিকা থেকে নির্বাচন করুন।
    4. 'বুদ্ধিবৃত্তিক সেবা এপিস' নির্বাচন করুন।
    5. "API Type" অপশনটি নির্বাচন করুন।
    6. "লেখা অনুবাদ" নির্বাচন করুন।
    7. 'স্তর মূল্য' বিভাগে 'মুক্ত স্তর (F0)"নির্বাচন করুন।
    8. বাকি আকারে বের হয়ে পূরণ করুন, এবং 'তৈরি করুন' বোতামটি চাপুন।
  4. আপনার "প্রত্যয়ন কী" উদ্ধার করা।
    1. 'সব সম্পদ ' যাও এবং Microsoft অনুবাদক API আপনার সদস্যতা নির্বাচন করুন।
    2. ' চাবি' অপশনে গিয়ে সার্ভিসটি ব্যবহার করতে আপনার সদস্যতা কী কপি।

সমস্যা সমাধান

কোনও অ্যাপে কোনো অ্যাপ্লিকেশন অনুবাদ করা হচ্ছে।

  • যদি কোনও অ্যাপ অনুবাদ, পরীক্ষা 'AllTrans ' Xposed ইনস্টলার মধ্যে সক্রিয় করা হয়। মডিউল ->.
  • এছাড়াও আপনার "Microsoft/ইয়ানডেক্স সাবস্ক্রিপশন কী' সঠিক কিনা পরীক্ষা করুন।

অনুবাদ অ্যাপ্লিকেশন উদ্বোধনী পর্দায় আটকে যায়।

  • অ্যাপের নাম 'অ্যাপস' অনুবাদ করতে ক্লিক করুন ট্যাব এবং ক্লিক করুন 'ক্লিক করার সুস্পষ্ট অনুবাদ ক্যাশ'।
  • এই প্রথমবারের মতো এই অ্যাপটি অনুবাদ করা শুরু হয়, অ্যাপটির নাম "অ্যাপ থেকে অনুবাদ" ট্যাব এবং 'দেরী আগে Replcaing সঙ্গে অনুবাদ করা লেখা' "অন্যান্য সেটিংস" পূরণ করুন ২০০০ করতে ক্লিক করুন।

অনুবাদ করলে অ্যাপ "বাহিনী বন্ধ করুন"।

  • অ্যাপের নাম 'অ্যাপস' অনুবাদ করতে ক্লিক করুন ট্যাব এবং ক্লিক করুন 'ক্লিক করার সুস্পষ্ট অনুবাদ ক্যাশ'।
  • অ্যাপটির নাম "অ্যাপ থেকে অনুবাদ" ট্যাবে ক্লিক করুন, এবং নিয়ে 'অন্য সেটিংস"'দেরী আগে Replcaing সঙ্গে অনুবাদ করা লেখা' ২০০০ থেকে ভর্তি।
  • অ্যাপের নাম 'অ্যাপস' অনুবাদ এ ক্লিক করুন ট্যাব এবং 'অন্য সেটিংস' ভর্তি 'দেরী আগে শুরু করা অনুবাদ WebViews মধ্যে' ২০০০ থেকে।

একটি বিশেষ অ্যাপ অনুবাদ করা হয়নি

  • অ্যাপের নাম 'অ্যাপস' অনুবাদ এ ক্লিক করুন ট্যাব এবং 'অন্য সেটিংস' ভর্তি 'দেরী আগে শুরু করা অনুবাদ WebViews মধ্যে' ২০০০ থেকে।
  • অ্যাপ্লিকেশন কিছু অংশ এখনও অনুবাদ করা হচ্ছে না, তবে অনুবাদ 'অ্যাপস' অ্যাপটির নাম ক্লিক করুন ট্যাব এবং "আক্রমনাত্মক মোড" সক্রিয় করা হবে।

গেম অ্যাপ অনুবাদ করা হয়নি

  • প্রযুক্তিগত সীমাবদ্ধতার কারণে অনেক গেম 'AllTrans' কাজ করবে না। তো এই সব করা যেতে পারে।

যদি তোমরা এখনও সমস্যা 'ঘনিষ্ঠ ফোর্স' বা অংশ অ্যাপ্লিকেশন অনুবাদ করা হচ্ছে না

সাহায্যের জন্য আমার সাথে যোগাযোগ করুন। যখন আপনি অন্তত নিম্নবর্তী তথ্য প্রয়োজন হবে-

  1. অ্যাপটি গুগল প্লে স্টোর, CoolApk, ইত্যাদি কিছু APK স্টোর থেকে অ্যাপ্লিকেশন ডাউনলোড করার লিংক সাথে নাম।
  2. আপনার ফোনে অ্যান্ড্রয়েড সংস্করণ।
  3. সমস্যা যা আপনি অ্যাপ ওপেন করার চেষ্টা করছেন, 'logcat'। আপনি অ্যাপস পাওয়া প্লে স্টোর থেকে, এই যেমন ব্যবহার করতে পারেন

আপনার উপরে আছে, একবার আমার সাথে যোগাযোগ XDA সূত্র সমর্থননিয়ে।
আপনি বিষয় গিটহাব প্রকল্পেরবিষয়ে রিপোর্ট করতে পারেন।
