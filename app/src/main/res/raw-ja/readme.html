AllTrans は

それは実行時に別の 1 つの言語から言語でアプリ内のすべてのテキストを置き換えます。

たとえばアプリはドイツ語と言います。ユーザーは、アプリケーション名、および必要な言語変換 (英語のドイツ語と言う) を選択します。

ユーザーは必要なアプリケーションを使用する際に、アプリでは、どこのすべてのテキストは、対応する英語に置き換えられます。

これは Chrome で Google 翻訳の仕組みのようです。

注- 技術的な制限のためこれは多くのゲームが動作しません。何もこのことについて行うことができます。

AllTrans を使用する方法

このアプリケーションに必要なMicrosoft または Yandex 翻訳からキーの無料サインアップします。キーを取得する方法については、以下の手順を参照してください。

Yandex 翻訳より多くの言語をサポートしている、はるかに簡単にサインアップ プロセスには、いいえクレジット カードが必要です。

マイクロソフト翻訳を与えるより良い翻訳、サインアップするクレジット カードが必要です。(しかし、何も心配しないで起訴)。

  1. AllTrans Xposed インストーラーでアプリを有効に -> モジュールを確認します。
  2. AllTrans アプリケーションを起動します。「グローバル設定」タブでには、Yandex を使用したい場合に"Yandex の代わりに Microsoft の使用"有効にします。Microsoft を使用する無効にします。
  3. 「言語から翻訳」や「翻訳言語」を選択します。
  4. 「翻訳するアプリ」タブでそれの横にあるチェック ボックスをクリックして、変換するアプリを見つけます。チェック ボックスが表示されない場合は、横向きに横にあなたの携帯電話を回転させてみてください。
  5. 翻訳しアプリを再起動を閉じて - それに翻訳する必要があります!

Yandex 変換キーを取得する方法

  1. Yandex 翻訳 API のウェブサイトへ
  2. アカウントが既にいない、「登録」をクリックし、新しいアカウントにサインアップします。
  3. サインアップした後、利用規約に同意して「API キーを取得」をクリックして。
  4. あなたのキーを受信しました!

サブスクリプション キーは"trnsl.1.1.201701..."のようなもの

Microsoft Azure 変換キーを取得する方法

  1. Microsoft Azure アカウントにサインアップします。
    Azure のアカウントがない場合は、 Microsoft Azure アカウントにサインアップします。
  2. アカウントを作成したら、 Azure ポータルにログインします。
  3. Azure アカウントに「Microsoft Translator API 購読」を追加します。
    1. 従量制のサブスクリプションを選択します。クレジット カードが必要になりますが、心配しないで、何もが課金されます。
    2. 「+ 新しい」オプションを選択します。
    3. サービスの一覧から「知性」を選択します。
    4. 「認知サービス Api」を選択します。
    5. 「API の種類」オプションをを選択します。
    6. 「テキスト翻訳」を選択します。
    7. 「価格層」セクションで「無料層 (F0)」を選択します。
    8. フォームの残りの部分に記入し、「作成」ボタンを押します。
  4. あなたの「認証キー」を取得します。
    1. 「すべてのリソース」に移動しを購読して Microsoft Translator API を選択します。
    2. 「キー」オプションに移動しサービスにアクセスするサブスクリプション キーをコピーします。

サブスクリプション キーは"321dcba..."のようなもの

トラブルシューティング

任意のアプリケーションでは、アプリケーションは変換されません。

  • 翻訳対象のアプリがない場合はチェックが Xposed インストーラーで"AllTrans"が有効になっているモジュールを ->。
  • 「マイクロソフト/Yandex サブスクリプション キー」が正しいかどうかも確認します。

オープニング画面で立ち往生している翻訳アプリ

  • 「翻訳アプリ」でアプリの名前をクリックして] タブをクリックし、"クリックしてする翻訳キャッシュをクリア」をクリックして。
  • 初めてこのアプリの翻訳を開始する場合は、2000 年に「翻訳するアプリ」タブで、「その他設定」で「遅延する前に Replcaing と翻訳テキスト」のアプリの名前をクリックしてします。

翻訳アプリは「強制終了」

  • 「翻訳アプリ」でアプリの名前をクリックして] タブをクリックし、"クリックしてする翻訳キャッシュをクリア」をクリックして。
  • 「翻訳するアプリ」タブでアプリの名前をクリックしてし、書き込む 2000年"遅延の前に Replcaing と翻訳テキスト"の「その他の設定」。
  • 「翻訳アプリ」でアプリの名前をクリックして] タブをクリックし、「その他設定」で 2000 に「遅延する前に開始する翻訳 web 表示」を入力。

特定のアプリが翻訳されていません。

  • 「翻訳アプリ」でアプリの名前をクリックして] タブをクリックし、「その他設定」で 2000 に「遅延する前に開始する翻訳 web 表示」を入力。
  • アプリのいくつかの部分は、まだ翻訳されていない場合、「に変換するアプリ」のアプリの名前をクリックして] タブをクリックし、、"積極的なモード"を有効にします。

ゲーム アプリが翻訳されていません。

  • 技術的な制限により多くのゲーム"AllTrans"は働かない。何もこのことについて行うことができます。

まだ翻訳されていないアプリの部分「強制終了」などの問題がある場合

サポートにご連絡します。少なくとも次の情報を必要がありますを行うと、

  1. グーグル プレイ ストア、CoolApk、等のようないくつかの APK のストアからアプリをダウンロードへのリンクとともに、アプリケーションの名前。
  2. あなたの携帯電話の Android のバージョン。
  3. あなたに問題を与えるアプリを開こうとするとあなたの「logcat」.このため、これといったプレイ店で自由に利用できるアプリを使用できます。

上記、 XDA のスレッドをサポートに連絡ください。
プロジェクトの Github上での問題を報告できます。
