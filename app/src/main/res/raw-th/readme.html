AllTrans อะไร

จะแทนที่ข้อความทั้งหมดในแอในภาษาจากภาษาหนึ่งไปยังอีกในขณะทำงาน

บอกตัวอย่างเช่น แอในเยอรมัน ผู้ใช้เลือกชื่อแอป และการแปลงภาษาต้อง (พูดภาษาเยอรมันเป็นภาษาอังกฤษ)

แล้ว เมื่อใดก็ ตามที่ผู้ใช้ใช้แอพที่จำเป็น ข้อความทั้งหมดได้ทุกที่ใน app จะถูกแทน โดยเทียบเท่าภาษาอังกฤษ

นี่คือสิ่งที่คล้ายกับวิธีการทำงานของ Google แปลภาษาใน Chrome

หมายเหตุ- เนื่องจากข้อจำกัดทางเทคนิค นี้จะไม่ทำงานกับเกมมากมาย ไม่มีอะไรสามารถทำได้เกี่ยวกับเรื่องนี้

วิธีการใช้ AllTrans

นี้โปรแกรมประยุกต์ต้องมีคุณสมัครฟรีจาก Microsoft หรือ Yandex แปล ดูคำแนะนำด้านล่างในการได้รับคีย์

แปล Yandex สนับสนุนภาษาเพิ่มเติม มีเป็นมากง่ายต่อการลงทะเบียน และต้องไม่มีบัตรเครดิต

Microsoft แปลให้แปลดีกว่าแต่ต้องใช้บัตรเครดิตเพื่อลงทะเบียน (แต่ไม่ต้องกังวล ไม่มีอะไรจะคิด.)

  1. ตรวจสอบให้แน่ใจว่า ได้เปิด app Installer ล Xposed AllTrans -> โมดู
  2. เปิดแอพ AllTrans ในแท็บ "การตั้งค่าสากล" เปิดใช้ "ใช้ Yandex แทนที่จะ Microsoft" ถ้าคุณต้องการใช้ Yandex ปิดใช้งานการใช้ Microsoft
  3. เลือก "แปลจากภาษา" และ "แปลภาษา"
  4. ในแท็บ "แอการแปล" ค้นหาแอพที่คุณต้องการแปล คลิกกล่องกาเครื่องหมายอยู่ถัดจาก ถ้ามองไม่เห็นกล่องกาเครื่องหมาย ลองหมุนไปด้านข้างของโทรศัพท์ในโหมดแนวนอน
  5. ปิด และรีสตาร์ทแอพที่ต้องการแปล - ควรแปล

วิธีรับคีย์แปล Yandex

  1. ไปYandex แปลเว็บไซต์ API
  2. ถ้าคุณไม่มีบัญชีผู้ใช้ คลิก "ลงทะเบียน" และลงชื่อสมัครใช้บัญชีใหม่
  3. หลังจากลงทะเบียน ยอมรับข้อตกลงผู้ใช้ และคลิก "รับ API Key"
  4. คุณได้รับรหัสของคุณเรียบร้อยแล้ว

คีย์บอกเป็นบางอย่างเช่น "trnsl.1.1.201701..."

วิธีรับคีย์แปลของ Microsoft Azure

  1. ลงทะเบียนสำหรับบัญชี Microsoft Azure
    ถ้าคุณไม่มีบัญชีผู้ใช้ Azure ลงทะเบียนสำหรับบัญชี Microsoft Azure
  2. หลังจากที่คุณมีบัญชีผู้ใช้ ลงเว็บไซต์ Azure
  3. เพิ่มเป็น "Microsoft แปล API สมัครสมาชิก" ในบัญชีของ Azure
    1. เลือกการบอกรับสมาชิก "จ่าย as-you-go" จะต้องมีบัตรเครดิต แต่ไม่ต้อง กังวล ไม่มีค่าธรรมเนียม
    2. เลือกตัวเลือก "+ ใหม่
    3. เลือก "ข้อมูลจากรายการของบริการ
    4. เลือก "APIs บริการองค์ความรู้"
    5. เลือกตัวเลือก "API ชนิด"
    6. เลือก "แปลข้อความ"
    7. ในส่วน "ระดับราคา" เลือก "ฟรีเทียร์ (F0)"
    8. กรอกส่วนที่เหลือของแบบฟอร์ม และกดปุ่ม
  4. ดึงข้อมูลของคุณ "กุญแจ"
    1. ไปที่ "ทรัพยากรทั้งหมด" และเลือก API แปล Microsoft ที่คุณได้สมัคร
    2. ไปที่ "คีย์" ตัวเลือก และคัดลอกคีย์สมัครสมาชิกของคุณเพื่อเข้าถึงบริการ

คีย์บอกเป็นบางอย่างเช่น "321dcba..."

การแก้ไขปัญหา

แอไม่มีการแปลในแอ

  • ถ้าไม่มี app คือการแปล การตรวจสอบการเปิดใช้งานตัวติดตั้งล Xposed "AllTrans" -> โมดู
  • ตรวจสอบว่า คีย์ของคุณ" Microsoft/Yandex บอก" ถูกต้อง

ติดอยู่ในหน้าจอเปิด app แปล

  • คลิกที่ชื่อแอใน "แอปที่แปล" แท็บ และคลิกที่ "คลิกเพื่อล้างแปลแค"
  • ถ้านี่เป็นครั้งแรกที่คุณเริ่มแปล app นี้ คลิกที่ชื่อแอ ในแท็บ "แอการแปล" และ "การหน่วงเวลาก่อนที่จะ Replcaing กับแปลข้อความ" กรอก "การตั้งค่าอื่นๆ" 2000

แอแปลเป็น "บังคับปิด"

  • คลิกที่ชื่อแอใน "แอปที่แปล" แท็บ และคลิกที่ "คลิกเพื่อล้างแปลแค"
  • คลิกที่ชื่อแอในแท็บ "แอการแปล" ใน "การตั้งค่าอื่นๆ" และกรอก และ "การหน่วงเวลาก่อนที่จะ Replcaing กับแปลข้อความ" 2000
  • คลิกที่ชื่อแอใน "แอปที่แปล" แท็บ และใน "การตั้งค่าอื่นๆ" กรอกข้อมูล "หน่วงเวลาก่อนที่จะเริ่มการแปล WebViews" 2000

ไม่มีการแปลแอปใด ๆ

  • คลิกที่ชื่อแอใน "แอปที่แปล" แท็บ และใน "การตั้งค่าอื่นๆ" กรอกข้อมูล "หน่วงเวลาก่อนที่จะเริ่มการแปล WebViews" 2000
  • หากบางส่วนของแอจะยังคงไม่มีการแปล คลิกที่ชื่อแอใน "แอปที่แปล" แท็บ และเปิดใช้งาน "โหมดก้าวร้าว"

ไม่มีการแปลเป็น app เกม

  • เนื่องจากข้อจำกัดทางเทคนิค "AllTrans" จะไม่ทำงานกับเกมมากมาย ไม่มีอะไรสามารถทำได้เกี่ยวกับเรื่องนี้

ถ้าคุณยังคงมีปัญหาเช่น "บังคับปิด" หรือส่วนของ app ไม่ถูกแปล

ติดต่อฉันสำหรับการสนับสนุน เมื่อคุณทำ จะต้องอย่างน้อยข้อมูลต่อไปนี้-

  1. ชื่อของ app พร้อมกับลิงค์เพื่อดาวน์โหลด appจากบางร้าน APK เช่น Google Play Store, CoolApk ฯลฯ
  2. ของโทรศัพท์ Android รุ่น
  3. "Logcat" ของคุณพยายามที่จะเปิดแอพซึ่งช่วยให้ปัญหา คุณสามารถใช้แอปพลิเคชันฟรีบน Play Store สำหรับนี้ เช่นนี้
