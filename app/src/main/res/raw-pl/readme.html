Co robi AllTrans

Zastępuje on cały tekst w aplikacji w języku z jednego języka na inny, w czasie wykonywania.

Powiedzieć, na przykład aplikacja jest w języku niemieckim. Użytkownik wybiera nazwę aplikacji i konwersji wymagane języka (powiedzmy z niemieckiego na angielski).

Następnie w każdym przypadku, gdy użytkownik korzysta z aplikacji wymagane, cały tekst, gdziekolwiek w aplikacji są zastępowane przez ich angielskie odpowiedniki.

Jest to coś podobnego do Google Translate działa w Chrome.

Uwaga - ze względu na techniczne ograniczenia, to nie będzie działać z wielu gier. Nic nie można zrobić o tym.

Jak używać AllTrans

Ta aplikacja wymaga aby zarejestrować się za darmo klucz od Microsoft lub tłumacza Yandex. Zobacz poniższe instrukcje, jak zdobyć klucze.

Tłumacza Yandex obsługuje więcej języków, ma dużo łatwiejszy proces rejestracji i wymaga karty kredytowej nr.

Microsoft Translate daje lepsze tłumaczenia, ale wymaga karty kredytowej, aby się zarejestrować. (Ale nie martw się, nic nie będzie opłata.)

  1. Upewnij się, że AllTrans jest włączona w instalatorze Xposed -> moduły.
  2. Uruchom aplikację AllTrans. W zakładce "Ustawienia globalne" po "Użyj Yandex instead of Microsoft", jeśli chcesz użyć Yandex. Wyłącz, aby użyć Microsoft.
  3. Wybierz "Tłumaczenie z języka" i "Tłumaczenie na język".
  4. W zakładce "Aplikacji do tłumaczenia" Znajdź aplikację, którą chcesz przetłumaczyć, kliknij pole wyboru obok niego. Jeśli pole wyboru nie jest widoczny, spróbuj, obracając telefon bokiem w orientacji poziomej.
  5. Zamknij i ponownie uruchom aplikację, którą chcesz przetłumaczyć - powinny być przetłumaczone!

Jak uzyskać klucz Tłumacz Yandex

  1. Przejdź do Yandex Translate API witryny
  2. Jeśli nie masz jeszcze konta, kliknij przycisk "Zarejestruj" i zarejestruj się na nowe konto.
  3. Po zarejestrowaniu się, zaakceptować umowę użytkownika i kliknij przycisk "Get API Key".
  4. Pomyślnie otrzymałeś klucz!

Klucz subskrypcji jest coś takiego jak "trnsl.1.1.201701..."

Jak uzyskać klucz tłumaczyć Microsoft Azure

  1. Zarejestruj się na konto Microsoft Azure.
    Jeśli nie masz już konto systemu Azure, zarejestruj się na koncie Microsoft Azure.
  2. Po utworzeniu konta, zaloguj się do Portalu Azure.
  3. Dodać "Subskrypcji Microsoft Translator API" do konta platformy Azure.
    1. Wybierz abonament "pay-as-you-go". Wymaga karty kredytowej, ale nie martw się, nic nie jest pobierana.
    2. Zaznacz opcję "+ Nowa".
    3. Wybierz "Inteligencji", z listy usług.
    4. Wybierz "Interfejsów API usługi poznawczych".
    5. Wybierz opcję "API typu".
    6. Wybierz "Tłumaczenie tekstu".
    7. W sekcji "Warstwy cenowej" Wybierz "Warstwa Bezpłatna (F0)".
    8. Wypełnij pozostałą część formularza i naciśnij przycisk "Utwórz".
  4. Pobrać klucz"Uwierzytelnianie".
    1. Przejdź do "Wszystkich zasobów" i wybierz opcję Microsoft Translator API, które subskrybujesz.
    2. Przejdź do opcji "Kluczy" i skopiować klucz subskrypcji na dostęp do usługi.

Klucz subskrypcji jest coś takiego jak "321dcba..."

Rozwiązywanie problemów

Żadna aplikacja nie jest tłumaczony w dowolnej aplikacji

  • Jeśli żadna aplikacja nie jest tłumaczony, wyboru jest "AllTrans" jest włączona w instalatorze Xposed -> moduły.
  • Należy również sprawdzić, jeśli "Klucz subskrypcji Microsoft/Yandex" jest poprawne.

Przetłumaczonych aplikacji tkwi w ekranie powitalnym

  • Kliknij nazwę aplikacji w "Aplikacje tłumaczenie" kartę i kliknij na "Kliknij aby wyczyścić tłumaczenie Cache".
  • Jeśli to Ci rozpoczął tłumaczenie tej aplikacji po raz pierwszy, kliknij nazwę aplikacji w zakładce "Aplikacji do tłumaczenia" i w "Inne ustawienia" wypełnić "Opóźnienie przed Replcaing z przetłumaczony tekst" do 2000.

Aplikacja tłumaczone jest "Życie Zamknij"

  • Kliknij nazwę aplikacji w "Aplikacje tłumaczenie" kartę i kliknij na "Kliknij aby wyczyścić tłumaczenie Cache".
  • Kliknij na nazwę aplikacji w zakładce "Aplikacji do tłumaczenia", i "Inne ustawienia" wypełnić "Opóźnienie przed Replcaing z przetłumaczony tekst" do 2000.
  • Kliknij nazwę aplikacji w "Aplikacje tłumaczenie" kartę, a w "Inne ustawienia" wypełnić "Opóźnienie przed rozpoczęciem do tłumaczenie WebViews", 2000.

Określonej aplikacji nie jest tłumaczony

  • Kliknij nazwę aplikacji w "Aplikacje tłumaczenie" kartę, a w "Inne ustawienia" wypełnić "Opóźnienie przed rozpoczęciem do tłumaczenie WebViews", 2000.
  • Jeśli niektóre części aplikacji nie są jeszcze przetłumaczone, kliknij nazwę aplikacji w "Aplikacje tłumaczenie" kartę, a następnie włączyć "Tryb agresywny".

Aplikacja gry nie jest tłumaczony

  • Z powodu ograniczeń technicznych "AllTrans" nie będzie działać z wielu gier. Nic nie można zrobić o tym.

Jeśli nadal masz problemy, takie jak "Życie Zamknij" lub części aplikacji nie są przetłumaczone

Poproś mnie o pomoc. Kiedy to zrobić, muszę conajmniej następujące informacje-

  1. Nazwa aplikacji, wraz z linkiem do pobrania aplikacji z niektóre APK sklep jak Sklep Google Play, CoolApk, itp.
  2. Twój telefon Android wersja.
  3. "Logcat" z was próbuje otworzyć aplikacji, które daje problemy. Można użyć swobodnie dostępnych aplikacji w Sklepie Play, takich jak ten jeden

Skoro masz powyżej, skontaktuj się ze mną na XDA obsługują wątku.
Problemy mogą również sprawozdanie w sprawie projektu Github.
