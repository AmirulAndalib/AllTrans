Τι σημαίνει AllTrans

Αντικαθιστά όλου του κειμένου σε μια εφαρμογή σε μια γλώσσα από τη μία γλώσσα στην άλλη κατά το χρόνο εκτέλεσης.

Να πω για παράδειγμα, μια εφαρμογή είναι στα γερμανικά. Ο χρήστης επιλέγει το όνομα της εφαρμογής, και η μετατροπή απαιτούμενες γλωσσικές (ας πούμε στα Γερμανικά σε Αγγλικά).

Στη συνέχεια, κάθε φορά που ο χρήστης χρησιμοποιεί την απαιτούμενη εφαρμογή, όλο το κείμενο, οπουδήποτε , στην εφαρμογή, αντικαθίστανται από Ελληνικά ισοδύναμά τους.

Αυτό είναι κάτι παρόμοιο με τον τρόπο λειτουργίας του Google Translate στο Chrome.

Σημείωση - λόγω τεχνικών περιορισμού, αυτό δεν θα λειτουργήσει με πολλά παιχνίδια. Τίποτα δεν μπορεί να γίνει σχετικά με αυτό.

Πώς μπορείτε να χρησιμοποιήσετε AllTrans

Αυτή η εφαρμογή απαιτεί να εγγραφείτε για δωρεάν ένα κλειδί από τη Microsoft ή Yandex μετάφραση. Δείτε τις παρακάτω οδηγίες για το πώς να πάρετε τα κλειδιά.

Μετάφραση Yandex υποστηρίζει περισσότερες γλώσσες, έχει μια πολύ ευκολότερη διαδικασία εγγραφής, και απαιτεί καμία πιστωτική κάρτα.

Microsoft Translate δίνει καλύτερες μεταφράσεις, αλλά απαιτεί μια πιστωτική κάρτα για να εγγραφείτε. (Αλλά μην ανησυχείτε, τίποτα δεν θα χρεωθεί.)

  1. Βεβαιωθείτε ότι AllTrans app είναι ενεργοποιημένη στο πρόγραμμα εγκατάστασης Xposed -> ενότητες.
  2. Ξεκινήστε την εφαρμογή AllTrans. Στην καρτέλα "Καθολικές ρυθμίσεις", μπορείτε να ενεργοποιήσετε «Χρήση Yandex αντί της Microsoft» αν θέλετε να χρησιμοποιήσετε Yandex. Απενεργοποίηση χρήσης Microsoft.
  3. Επιλέξτε το «Μεταφράσουν από γλώσσα» και «Μεταφράσουν γλώσσα».
  4. Στην καρτέλα «Εφαρμογές να μεταφράσουν», βρείτε την εφαρμογή που θέλετε να μεταφράσετε, επιλέξτε το πλαίσιο ελέγχου δίπλα σε αυτό. Εάν το πλαίσιο ελέγχου δεν είναι ορατό, δοκιμάστε περιστρεφόμενο πλαγίως το τηλέφωνό σας σε οριζόντιο προσανατολισμό.
  5. Κλείστε και επανεκκινήστε την εφαρμογή που θέλετε να μεταφραστούν - θα πρέπει να μεταφραστεί!

Πώς να πάρει Yandex μεταφράσει κλειδί

  1. Πηγαίνετε στο Yandex μεταφράσετε ιστοσελίδα API
  2. Εάν δεν έχετε ήδη ένα λογαριασμό, κάντε κλικ στο κουμπί «εγγραφή» και εγγραφείτε για ένα νέο λογαριασμό.
  3. Μετά την εγγραφή σας, συμφωνείτε με τους όρους χρήσης και κάντε κλικ στο κουμπί «Πάρει API κλειδί».
  4. Έχετε λάβει επιτυχώς το κλειδί σας!

Η κλείδα είναι κάτι σαν «trnsl.1.1.201701...»

Πώς να μεταφράσει κλειδί του Microsoft Azure

  1. Εγγραφείτε για ένα λογαριασμό Microsoft Azure.
    Αν δεν έχετε ήδη ένα λογαριασμό σας Azure, εγγραφείτε για ένα λογαριασμό Microsoft Azure.
  2. Αφού έχετε ένα λογαριασμό, συνδεθείτε με την Πύλη Azure.
  3. Για να προσθέσετε ένα «συνδρομή API Microsoft Μεταφραστής» στο λογαριασμό σας Azure.
    1. Επιλέξουν μια συνδρομή για «pay-as-you-go». Θα απαιτήσει μια πιστωτική κάρτα, αλλά μην ανησυχείτε, τίποτα δεν θα χρεωθεί.
    2. Επιλέξτε την επιλογή "+ νέα".
    3. Επιλέξτε «Νοημοσύνη» από τη λίστα των υπηρεσιών.
    4. Επιλέξτε «API γνωστική υπηρεσίες».
    5. Επιλέξτε την επιλογή «API τύπου».
    6. Επιλέξτε «Μετάφραση κειμένου».
    7. Στην ενότητα «Τιμολόγηση βαθμίδας», επιλέξτε το «δωρεάν βαθμίδα (F0)».
    8. Συμπληρώστε την υπόλοιπη φόρμα και πατήστε το κουμπί «Δημιουργία».
  4. Για να ανακτήσετε σας» κλειδί ελέγχου ταυτότητας».
    1. Πηγαίνετε σε «Όλους τους πόρους» και επιλέξτε το μεταφραστή API Microsoft που έχετε εγγραφεί σε.
    2. Πηγαίνετε στην επιλογή «Τα κλειδιά» και αντιγράψτε το κλειδί εγγραφής για πρόσβαση στην υπηρεσία.

Αντιμετώπιση προβλημάτων

Υπάρχει εφαρμογή αυτή μεταφράζεται σε οποιαδήποτε εφαρμογή

  • Αν δεν υπάρχει app is being μετέφρασα, έλεγχος είναι «AllTrans» είναι ενεργοποιημένη στο πρόγραμμα εγκατάστασης Xposed -> ενότητες.
  • Επίσης ελέγξτε αν σας «κλειδί συνδρομή Microsoft/Yandex» είναι σωστή.

Μετάφραση app είναι κολλημένο στην οθόνη έναρξης

  • Κάντε κλικ στο όνομα της εφαρμογής στο «Apps να μεταφράσει» καρτέλα και κάντε κλικ στο «Κάντε κλικ στο κουμπί να σαφές Cache μετάφραση».
  • Εάν αυτό είναι την πρώτη φορά που ξεκινήσατε τη μετάφραση αυτό το app, κάντε κλικ στο όνομα της εφαρμογής στην καρτέλα «Εφαρμογές να μεταφράσουν» και «Άλλες ρυθμίσεις» συμπληρώστε «Καθυστέρηση πριν από την Replcaing με μεταφρασμένο κείμενο» 2000.

Η εφαρμογή μετάφραση είναι «Κλείσιμο Force»

  • Κάντε κλικ στο όνομα της εφαρμογής στο «Apps να μεταφράσει» καρτέλα και κάντε κλικ στο «Κάντε κλικ στο κουμπί να σαφές Cache μετάφραση».
  • Κάντε κλικ στο όνομα της εφαρμογής στην καρτέλα «Εφαρμογές να μεταφράσουν» και σε «άλλες ρυθμίσεις» συμπληρώστε «Καθυστέρηση πριν από την Replcaing με μεταφρασμένο κείμενο» έως το 2000.
  • Κάντε κλικ στο όνομα της εφαρμογής στο «Apps να μεταφράσει» καρτέλα, και στις «Άλλες ρυθμίσεις» συμπληρώστε «Καθυστέρηση πριν αρχίζει να μεταφράσει WebViews» έως το 2000.

Μια συγκεκριμένη εφαρμογή δεν is being μετέφρασα

  • Κάντε κλικ στο όνομα της εφαρμογής στο «Apps να μεταφράσει» καρτέλα, και στις «Άλλες ρυθμίσεις» συμπληρώστε «Καθυστέρηση πριν αρχίζει να μεταφράσει WebViews» έως το 2000.
  • Εάν ορισμένα τμήματα της εφαρμογής εξακολουθούν να μην μεταφράζονται, κάντε κλικ στο όνομα της εφαρμογής στο «Apps να μεταφράσει» καρτέλα, και να ενεργοποιήσετε την «Επιθετική λειτουργία».

Ένα παιχνίδι app είναι δεν μεταφράζονται

  • Λόγω τεχνικών περιορισμών, «AllTrans» δεν θα λειτουργήσει με πολλά παιχνίδια. Τίποτα δεν μπορεί να γίνει σχετικά με αυτό.

Αν εξακολουθείτε να έχετε προβλήματα, όπως «Κλείσιμο ισχύ» ή τμήματα μιας εφαρμογής που δεν μεταφράζονται
