AllTrans mitä

Se korvaa kaiken tekstin app kielestä toiseen suorituksen.

Sanoa esimerkiksi app saksaksi. Käyttäjä valitsee app nimi ja kieli muuntaminen (eli saksaksi ja Englanti).

Sitten aina, kun käyttäjä käyttää tarvittavat app, kaikki teksti ANYWHERE -ohjelmassa korvataan heidän Englanti vastaavien.

Tämä on jotain vastaavaa Google Translate toimintatapaa Chrome.

Huomautus - teknisiä rajoituksia, koska tämä ei toimi monia pelejä. Mitään ei tehdä tästä.

Kuinka käyttää AllTrans

Tämä sovellus edellyttää te jotta allekirjoittaa jalkeilla ajaksi vapauttaa avain tai Yandex Käännä. Katso alla olevia ohjeita saat avaimet.

Yandex Käännä kielelle on helpompi kirjautumisen yhteydessä, ja vaatii n luottokortin.

Microsoft Translate antaa parempia käännöksiä, mutta vaatii luottokortin rekisteröityä. (Mutta älä huoli, mikään ei ole hintaan.)

  1. Varmista, että AllTrans app on käytössä Xposed Installer -> Modules.
  2. Käynnistää AllTrans app. Ota ”Global asetukset”, ”käytä Yandex sijaan Microsoft”, jos haluat käyttää Yandex. Poista Käytä Microsoft.
  3. Valitse ”Käännä kielestä” ja ”kääntää kieli”.
  4. ”Apps kääntää”-välilehti Etsi sovellus, jonka haluat kääntää, valitse sen vieressä oleva valintaruutu. Jos valintaruutu ei ole näkyvissä, kokeile puhelinta sivuttain kääntymässä vaaka-asennossa.
  5. Sulje ja Käynnistä sovellus käännettävä - olisi käännettävä!

Miten saada Yandex kääntää avain

  1. Siirry Yandex Käännä API Website
  2. Jos sinulla ei vielä ole tiliä, Klikkaa ”Rekisteröidy” ja kirjaudu uusi tili.
  3. Kirjautumisen jälkeen Hyväksy käyttöehdot ja valitse ”saada API-avain”.
  4. Olet saanut avain!

Tilauskoodi on jotain ”trnsl.1.1.201701...”

Miten saada Microsoftin Azure kääntää avain

  1. Rekisteröidy Microsoftin Azure-tili.
    Jos sinulla ei vielä ole Azure tiliä, Rekisteröidy Microsoftin Azure huomioon.
  2. Kun sinulla on tili, Kirjaudu Azure portaali.
  3. Lisää ”Microsoft Kääntäjä API tilaus” Azure tiliisi.
    1. Valitse ”pay-as-you-go” tilaus. Edellyttää luottokorttia, mutta älä huoli, mitään ei veloiteta.
    2. ”+ Uusi”-vaihtoehdon.
    3. Valitse palveluluettelosta ”älykkyys”.
    4. Valitse ”kognitiivinen Palvelut APIs”.
    5. ”API Type”-vaihtoehdon.
    6. Valitse ”Lähetä”.
    7. Valitse ”hinnoittelu Tier”-osassa ”Ilmainen Tier (F0)”.
    8. Täytä lomake loppuun ja paina ”Luo”-painiketta.
  4. Hakea ”ohjelmistoavaimesi”.
    1. Mene ”kaikki resurssit” ja valitse Microsoft Kääntäjä API olet tilannut.
    2. Mene ”avaimet”-vaihtoehto ja kopioi tilauskoodisi palvelua.

Tilauskoodi on jotain ”321dcba...”

Vianmääritys

Ole app. on käännetty tahansa App

  • Jos ei ole app ei käännetä, uloskirjautuminen on ”AllTrans” on käytössä Xposed Installer -> Modules.
  • Myös tarkistaa, jos ”Microsoft/Yandex tilauskoodisi” on oikea.

Käännetty app on juuttunut aloitusnäyttö

  • Napsauta sovelluksen nimeä ”Apps kääntää” välilehti ja klikkaa ”Valitse to Clear käännös Cache”.
  • Jos tämä on ensimmäinen kerta alkoi kääntää tämän app, klikkaa app nimi ”Apps kääntää” välilehti ja ”muut asetukset” täyttö ”viive ennen Replcaing kanssa käännetty teksti” 2000.

Kääntäjä sovellus on ”Force Close”

  • Napsauta sovelluksen nimeä ”Apps kääntää” välilehti ja klikkaa ”Valitse to Clear käännös Cache”.
  • Napsauta sovelluksen nimeä ”Apps kääntää” välilehti ja kaupungissa ”muut asetukset” täyttämällä ”viive ennen Replcaing kanssa käännetty teksti” 2000.
  • Napsauta sovelluksen nimeä ”Apps kääntää” välilehti ja ”muut asetukset” täyttämällä ”viive ennen aloittamista voit kääntää Web-näkymä” 2000.

Erityisesti app ei ole käännetty

  • Napsauta sovelluksen nimeä ”Apps kääntää” välilehti ja ”muut asetukset” täyttämällä ”viive ennen aloittamista voit kääntää Web-näkymä” 2000.
  • Jos osa app ovat edelleen ei käännetä, klikkaa app nimi ”Apps kääntää” välilehti ja ”aggressiivinen tila”.

Peli app ei ole käännetty

  • Johtui teknisistä rajoituksista ”AllTrans” ei toimi monia pelejä. Mitään ei tehdä tästä.

Jos sinulla on vielä ongelmia, kuten ”Force Close” tai osa app ei käännetä

Minuun tukea. Kun teet, tarvitsen ainakin seuraavat tiedot-

  1. Nimi app, sekä linkki ladata app APK-kaupasta, kuten Google Soita Store, CoolApk, jne.
  2. Puhelimen Android versio.
  3. ”Logcat” yrität Avaa sovellus, joka antaa sinulle ongelmia. Voit käyttää apps vapaasti saatavilla Play Storesta, kuten Tämä
