Ba'ax todotransporte

Sustituye tuláakal le analte'o' ti' jump'éel ka'anatako'ob ti' jump'éel lenguaje jump'éel t'aano' ti' uláak' tu k'iinil meyajta'ale'.

A'al je'ebix jump'éel ka'anatako'ob jach tu alemán. Juntúul usuario selecciona u k'aaba' le ka'anatako'ob yéetel le tzeltalo'obo' le t'aano' (digamos u alemán u ingles).

Túun pakteche' kin le usuario meyajtiko'ob le ka'anatako'ob k'a'abeto'ob, tuláakal le analte'o', ANYWHERE ti' le ka'anatako'ob, ku sustituidos tumen u equivalentes ich inglés.

Le ba'ala' es ba'al similar ti' u páajtalil ti' u traducir u Google ku meyaj ich Chrome.

Nota - debido a jump'éel limitación técnica, le je'ela' mixtan u meyaj yéetel ya'ab juegos. Mix ba'al ku je'el u meentik yóok'ol le ba'ala'.

Bix utilizar todotransporte

Le ka'anatako'ob k'a'abet u pida gratis jump'éel clave ti' Microsoft wa Yandex traducir. Wilo'obe' le t'aano'ob tu continuación yóok'ol bix kéen p'áatak le claves.

Traducir Yandex soporta asab t'aano'ob, yaan jump'éel tuukula' registro jach asab ch'a'abil yéetel k'a'abet tarjeta p'aax MA'.

Translate Microsoft ku ts'aik le ti' traducciones, ba'ale' k'a'abet jump'éel tarjeta p'aax utia'al u inscribir. (Ba'ale' Ma'chen, mixba'al yaan u beel cargado.).

  1. Asegurar ti' u ka'anatako'ob táan habilitada ti' le instalador Xposed todotransporte-> módulos.
  2. Lanzar ka'anatako'ob todotransporte. Ti' le ju'unil tu'ux "Configuración Global", activar "Utilizar Yandex ti' lugar u Microsoft" wa k'áato' utilizar Yandex. Desactivar utia'al u utilizar Microsoft.
  3. Elige le "Traducir ti' le lenguaje" yéetel "Traducir le t'aano'".
  4. U ju'unil tu'ux "Aplicaciones ti' traducir", kaxtik yo'olal ba'ax k'áato' traducir, beetik clic ti' le casilla verificación yiknal leti'. Wa u casilla verificación ma' le visible, trate u girar le sutuchi'ilt'aan ti' le tu'ux u modo ma' chéen óolal.
  5. Mak yéetel reinicie yo'olal u k'áato' traducido - k'a'ana'an u traducido!

Bix kéen p'áatak clave u traducir u Yandex

  1. Bin Yandex traducir Web API
  2. Wa mina'an teech jump'éel yilaje', u beetik clic ti' "registrar u" ka registrar ti' jump'éel túumben yilaje'.
  3. Ka' registrar u, táan de acuerdo con le usuario ka beetik clic ti' "Kéen p'áatak le API Key".
  4. Ts'o'ok k'a'am belal u clave!

Le clave suscripción le ba'al bey "trnsl.1.1.201701..."

Bix kéen p'áatak clave u traducir Microsoft Azure

  1. Registrar a ti' jump'éel yilaje' Microsoft Azure.
    Wa mina'an teech jump'éel yilaje' Azure, registrar a ti' jump'éel yilaje' Azure u Microsoft.
  2. Ka' ts'o'ok u yaantal jump'éel yilaje', registrar a ti' le Portal ku Azure.
  3. Añadir jump'éel "suscripción API Microsoft traductor" ti' tu yilaje' ch'ooj.
    1. Elegir jump'éel suscripción u "prepago". Requerirá jump'éel tarjeta p'aax, ba'ale' tu Ma'chen, ma' u cargará.
    2. Seleccione le opción "+ túumben".
    3. Seleccione "Na'at" u le tsoolol áantajo'ob.
    4. Seleccione "APIs yéetel cognitivos".
    5. Seleccione le opción "Bin yano'ob ti' API".
    6. Seleccione "Traducción analte'o'".
    7. Ti' le sección "Nivel u tojol", seleccione le "Free Tier (F0)".
    8. Llene u yaalab le formulario yéetel presione le botón "Crear".
  4. "Recuperar u clave autenticación ti' le" "."
    1. Xi'iken "Tuláakal le nu'ukulo'ob" ka seleccione u API le traductor Microsoft tech suscrito ti'.
    2. Bin u opción "Claves" yéetel copiar le clave suscripción utia'al tsáabaltio'ob áantaj le mayaj.

Le clave suscripción le ba'al bey "321dcba..."

Solución toop

Yo'olal ma' u táan traduciendo ti' je'el ka'anatako'ob

  • Wa yo'olal ma' u táan traduciendo, cheque le "Todotransporte" Táan habilitada ti' le instalador Xposed-> módulos.
  • Compruebe xan wa u "clave u suscripción Microsoft leti' Yandex" Jach ma'alob.

Ka'anatako'ob traducida táan atascada ti' le pantalla Chúunul

  • Beetik clic ti' k'aaba' le ka'anatako'ob ti' "Aplicaciones utia'al traducir" ficha ka beetik clic ti' "beetik clic ti' u clara traducción caché".
  • Wa táan u yáax ka'atéen empezaste traducir le ka'anatako'ob, beetik clic ti' k'aaba' le ka'anatako'ob tu táan "Aplicaciones ti' traducir" yéetel ti' "La ajustes" rellene "Retraso bey ma' Replcaing" yéetel traducir analte'o' utia'al 2000.

Le ka'anatako'ob traducido le "muuk'"

  • Beetik clic ti' k'aaba' le ka'anatako'ob ti' "Aplicaciones utia'al traducir" ficha ka beetik clic ti' "beetik clic ti' u clara traducción caché".
  • Beetik clic ti' k'aaba' le ka'anatako'ob tu táan "Aplicaciones ti' traducir", ka tu ti' "la ajustes" rellene "Retraso bey ma' Replcaing" yéetel traducir analte'o' ti' le 2000.
  • Beetik clic ti' k'aaba' le ka'anatako'ob ti' "Aplicaciones utia'al traducir" ficha yéetel ti' "La ajustes" rellene "Retraso bey ma' comenzar traducir WebViews" ti' le 2000.

Jump'éel ka'anatako'ob t'aana' ku ma' u táan traduciendo

  • Beetik clic ti' k'aaba' le ka'anatako'ob ti' "Aplicaciones utia'al traducir" ficha yéetel ti' "La ajustes" rellene "Retraso bey ma' comenzar traducir WebViews" ti' le 2000.
  • Wa Jayp'éel pak'chaje' le ka'anatako'ob láayli' u táan traduciendo ma', beetik clic ti' k'aaba' le ka'anatako'ob ti' "Aplicaciones utia'al traducir" ficha ka activar le "Modo agresivo".

Jump'éel ka'anatako'ob báaxal ma' u táan traduciendo

  • Debido a limitaciones técnicas "Todotransporte" mixtan u meyaj yéetel ya'ab juegos. Mix ba'al ku je'el u meentik yóok'ol le ba'ala'.

Wa láayli' yaan talamilo'ob bey le "Force Close" wa pak'chaje' jump'éel ka'anatako'ob ma' u tsolik ichil

  1. U k'aaba' le ka'anatako'ob, junto yéetel jump'éel enlace utia'al descargar yo'olal u Jayp'éel koonol APK bey Google Play Store, CoolApk, etcetera.
  2. Versión Android ti' le nu'ukul t'aan.
  3. Jump'éel "logcat" u k'aax u Je'e yo'olal ba'ax ku ts'aik ti' talamilo'ob. Je'el utilizar aplicaciones disponibles ti' Play Store utia'al le ba'ala', bey le
