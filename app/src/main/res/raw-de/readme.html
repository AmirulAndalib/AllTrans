Was bedeutet AllTrans

Der gesamte Text in einer app in einer Sprache von einer Sprache in eine andere zur Laufzeit ersetzt.

Sagen Sie zum Beispiel eine app in deutscher Sprache ist. Ein Benutzer wählt den app-Namen und die gewünschte Sprachkonvertierung (sagen Deutsch-Englisch).

Dann wenn der Benutzer die gewünschte app verwendet, der gesamte Text, überall in der app sind ersetzt durch deren englische Entsprechung.

Dies ist etwas ähnlich wie Google Translate in Chrome funktioniert.

Hinweis : aufgrund einer technischen Limitierung, dies mit vielen Spielen funktioniert nicht. Nichts kann dazu tun.

Gewusst wie: verwenden Sie AllTrans

Diese Anwendung erfordert Sie einen Schlüssel von Microsoft oder Yandex Translate kostenlos anmelden. Finden Sie die folgenden Anweisungen um den Schlüssel zu bekommen.

Yandex Translate unterstützt mehrere Sprachen, hat ein viel einfacher Anmeldevorgang und erfordert keine Kreditkarte.

Microsoft Translate gibt bessere Übersetzungen, aber erfordert eine Kreditkarte zu registrieren. (Aber keine Sorge, nichts wird sein berechnet.)

  1. Stellen Sie sicher, AllTrans app im Xposed Installateur aktiviert ist -> Module.
  2. Starten Sie AllTrans-app. "Globaleinstellungen" Registerkarte "aktivieren Sie"Verwenden Yandex Instead Of Microsoft"Yandex verwenden möchten. Deaktivieren, um Microsoft zu verwenden.
  3. Wählen Sie "Übersetzen von Sprache" und "Sprache übersetzen".
  4. Finden Sie im Reiter "Apps zu übersetzen" die app zu übersetzen, klicken Sie auf das Kontrollkästchen neben dem gewünschten. Wenn das Kontrollkästchen nicht angezeigt wird, versuchen Sie, Ihr Telefon seitwärts ins Querformat drehen.
  5. Schließen und starten Sie die app, die übersetzt werden soll - es sollte übersetzt werden!

Gewusst wie: Yandex übersetzen Schlüssel zu bekommen

  1. Gehen Sie auf Yandex API Website übersetzen
  2. Wenn Sie bereits ein Konto haben, klicken Sie auf "registrieren" und melden Sie sich für ein neues Konto.
  3. Verpflichten Sie nach der Anmeldung sich die Benutzervereinbarung und klicken Sie auf "Get API Key".
  4. Sie haben erfolgreich Ihre Schlüssel erhalten!

Die Abonnementschlüssel ist so etwas wie "trnsl.1.1.201701..."

Gewusst wie: Microsoft Azure übersetzen Schlüssel zu bekommen

  1. Melden Sie sich für ein Microsoft Azure-Konto.
    Wenn Sie bereits ein Azure-Konto haben, melden Sie sich für ein Microsoft Azure-Konto.
  2. Nachdem Sie ein Konto haben, melden Sie sich bei der Azure-Portal.
  3. Ein "Microsoft Translator API-Abonnement" zu Ihrem Azure Konto hinzufügen.
    1. Wählen Sie ein "Pay as you go"-Abo. Benötigen Sie eine Kreditkarte, aber keine Sorge, nichts berechnet werden.
    2. Wählen Sie die Option "+ neu".
    3. Wählen Sie "Intelligenz" aus der Liste der Dienste.
    4. Wählen Sie "Kognitive Services-APIs".
    5. Wählen Sie die Option "API".
    6. Wählen Sie "Textübersetzung".
    7. Wählen Sie im Abschnitt "Preise-Tier" der "Free-Tarif (F0)".
    8. Füllen Sie den Rest des Formulars, und drücken Sie die Schaltfläche "Erstellen".
  4. Rufen Sie Ihre "Authentifizierungs-Schlüssel".
    1. Gehen Sie auf "Alle Ressourcen" und wählen Sie Microsoft Translator API, die Sie abonniert haben.
    2. Gehen Sie auf die Option "Schlüssel" und kopieren Sie Ihren Abonnementschlüssel auf den Dienst zuzugreifen.

Die Abonnementschlüssel ist so etwas wie "321dcba..."

Fehlerbehebung

Keine app wird in jeder app übersetzt

  • Wenn keine app übersetzt wird, Check ist "AllTrans" in Xposed Installer aktiviert ist -> Module.
  • Auch prüfen Sie, ob Ihre "Microsoft/Yandex Abonnementschlüssel" korrekt ist.

Übersetzte app ist im Eröffnungsbildschirm stecken.

  • Klicken Sie auf den Namen der app im "Apps zu übersetzen" Registerkarte, und klicken Sie auf "Klicken Sie auf Übersetzung Cache löschen".
  • Wenn dies das erste Mal Sie angefangen ist, diese app zu übersetzen, klicken Sie auf den Namen der app auf "Apps zu übersetzen" Registerkarte "und in"Andere Einstellungen"ausfüllen"Verzögerung vor Replcaing mit übersetzt Text"bis 2000.

Die Translated app ist "Force Close"

  • Klicken Sie auf den Namen der app im "Apps zu übersetzen" Registerkarte, und klicken Sie auf "Klicken Sie auf Übersetzung Cache löschen".
  • Klicken Sie auf den Namen der app auf "Apps zu übersetzen" Registerkarte "und in"andere Einstellungen"ausfüllen"Verzögerung vor Replcaing mit übersetzt Text"bis 2000.
  • Klicken Sie auf den Namen der app im "Apps zu übersetzen" Registerkarte, und unter "Andere Einstellungen" ausfüllen "Verzögerung vor dem Start zu übersetzen WebViews" bis 2000.

Eine bestimmte app wird nicht übersetzt

  • Klicken Sie auf den Namen der app im "Apps zu übersetzen" Registerkarte, und unter "Andere Einstellungen" ausfüllen "Verzögerung vor dem Start zu übersetzen WebViews" bis 2000.
  • Wenn einige Teile der app noch nicht übersetzt sind, klicken Sie auf den Namen der app im "Apps zu übersetzen" Registerkarte, und aktivieren Sie "Aggressive Mode".

Eine Spiele-app wird nicht übersetzt

  • Aufgrund technischer Einschränkungen funktioniert nicht "AllTrans" mit vielen spielen. Nichts kann dazu tun.
