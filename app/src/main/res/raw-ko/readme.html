AllTrans는

런타임에 다른 하나의 언어에서 언어에 응용 프로그램에서 모든 텍스트 를 대체합니다.

라고 예를 들어 응용 프로그램 독일어. 사용자는 응용 프로그램 이름 및 필요한 언어 변환 (말 영어 독일어)를 선택합니다.

다음 때마다 사용자 필요한 애플 리 케이 션을 사용 하 여, 모든 텍스트, ANYWHERE 는 애플 리 케이 션에 상응 하는 영어에 의해 대체 됩니다.

이것이 Google 번역 크롬에서 작동 하는 방법은 비슷합니다.

참고 -기술적인 제한 때문에이 작동 하지 않습니다 많은 게임. 아무것도 이것에 대해 할 수 있습니다.

AllTrans를 사용 하는 방법

이 응용 프로그램에 필요한 Microsoft 또는 Yandex 번역에서 키 무료 가입을. 키를 얻는 방법에 아래 지침을 참조 하십시오.

Yandex 번역 더 많은 언어를 지 원하는, 훨씬 쉽게 가입 프로세스, 및 아니오 신용 카드필요.

마이크로소프트 번역 준다 더 나은 번역, 그러나 가입 신용 카드를 요구 한다. (하지만 걱정 하지 마세요, 아무것도 됩니다 청구.)

  1. AllTrans 애플 리 케이 션 Xposed 설치 프로그램에서 활성화-> 단위 다는 것을 확인 하십시오.
  2. AllTrans 응용 프로그램을 시작 합니다. Yandex를 사용 하려는 경우 "전역 설정" 탭에서 "Yandex 대신 Microsoft 사용"을 사용 합니다. Microsoft를 사용 하 여 비활성화 합니다.
  3. 선택 "번역 언어 에서"와 "번역 언어".
  4. "번역 애플 리 케이 션" 탭에서 번역, 옆에 있는 확인란을 클릭 하 고 싶은 응용 프로그램을 찾습니다. 확인란 표시 되지 않는 경우에, 당신의 전화를 옆으로 가로 모드로 회전 보십시오.
  5. 닫고 다시 시작 애플 리 케이 션 당신이 원하는 번역-번역 되어야 한다!

Yandex 번역 키를 얻으려면 어떻게

  1. Yandex 번역 API 웹사이트에 가 서
  2. 계정이 없는 경우 "등록"을 클릭 하 고 새 계정에 가입.
  3. 가입 후, 사용자 계약에 동의 하 고 "" API 키를 얻을 클릭 합니다.
  4. 키를 성공적으로 받은!

Microsoft Azure 번역 키를 얻으려면 어떻게

  1. Microsoft Azure 계정에 가입.
    하늘빛 계정이 없는 경우 Microsoft Azure 계정에 가입 하십시오.
  2. 계정을 Azure 포털에 로그인 합니다.
  3. 하늘빛 계정 "마이크로소프트 번역 API 구독"을 추가 합니다.
    1. "종 량 제" 구독을 선택 합니다. 신용 카드 필요 합니다 하지만 걱정 하지 마세요 아무것도 부과 됩니다.
    2. "+ 새" 옵션을 선택 합니다.
    3. 서비스 목록에서 "정보"를 선택 합니다.
    4. "인지 서비스 Api"를 선택 합니다.
    5. "API 유형" 옵션을 선택 합니다.
    6. "텍스트 번역"을 선택 합니다.
    7. "가격 계층" 섹션에서 "무료 계층 (F0)"을 선택 합니다.
    8. 폼의 나머지를 "만들기" 버튼을 누릅니다.
  4. "인증 키"를 검색 합니다.
    1. "모든 자원"로 이동 하 고 Microsoft 번역기 API에 가입을 선택 합니다.
    2. "키" 옵션에가 고 서비스에 액세스 하 여 구독 키 복사.

문제 해결

아니 애플 리 케이 션 응용 프로그램에 번역 되 고

  • 아무 애플 리 케이 션 번역 되 고, 경우 확인은 "AllTrans" Xposed 설치 프로그램에서 활성화-> 단위.
  • 또한 "마이크로소프트/Yandex 구독 키"를 올바른지 확인 하십시오.

번역된 애플 리 케이 션 시작 화면에 갇혀

  • "번역 애플 리 케이 션"에서 응용 프로그램의 이름을 클릭 하 여 탭, 및 "클릭을 번역 캐시 지우기"를 클릭 합니다.
  • 처음으로 번역 하는이 애플 리 케이이 션 시작 이면 2000 응용 프로그램의 이름을 "번역 애플 리 케이 션" 탭, 그리고 "지연 전에 Replcaing와 번역 텍스트"에서 "기타 설정" 채우기를 클릭 합니다.

번역 애플 리 케이 션은 "강제로 닫기"

  • "번역 애플 리 케이 션"에서 응용 프로그램의 이름을 클릭 하 여 탭, 및 "클릭을 번역 캐시 지우기"를 클릭 합니다.
  • "번역 애플 리 케이 션" 탭에서 응용 프로그램의 이름을 클릭 하 고에서 "기타 설정" 2000 "지연 전에 Replcaing와 번역 텍스트" 입력.
  • "번역 애플 리 케이 션"에서 응용 프로그램의 이름을 클릭 하 여 탭, 및 "기타 설정"에서 작성 "지연 하기 전에 시작 하기 번역 WebViews" 2000.

특정 애플 리 케이 션 하지 번역 되 고

  • "번역 애플 리 케이 션"에서 응용 프로그램의 이름을 클릭 하 여 탭, 및 "기타 설정"에서 작성 "지연 하기 전에 시작 하기 번역 WebViews" 2000.
  • 응용 프로그램의 일부는 여전히 하지 번역 되 고, "번역 애플 리 케이 션"에서 응용 프로그램의 이름을 클릭 합니다 탭, 그리고 "적극적인 모드"를 사용.

게임 애플 리 케이 션 하지 번역 되 고

  • 기술적인 제한으로 인해 많은 게임 "AllTrans" 작동 하지 않습니다. 아무것도 이것에 대해 할 수 있습니다.

아직도 "강제로 닫기" 또는 번역 되지 응용 프로그램의 일부와 같은 문제가 있는 경우
