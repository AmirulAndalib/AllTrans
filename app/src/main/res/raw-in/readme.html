Apakah AllTrans

Ini menggantikan semua teks di app dalam bahasa dari satu bahasa lain pada saat runtime.

Mengatakan misalnya app adalah dalam bahasa Jerman. Pengguna memilih nama app, dan konversi bahasa yang diperlukan (mengatakan Jerman ke bahasa Inggris).

Kemudian setiap kali pengguna menggunakan app diperlukan, Semua teks, ANYWHERE di app, digantikan oleh Inggris setara.

Ini adalah sesuatu yang mirip dengan cara kerja Google Terjemahan di Chrome.

Catatan - karena pembatasan teknis, ini tidak akan bekerja dengan banyak permainan. Tidak ada yang dapat dilakukan tentang hal ini.

Bagaimana menggunakan AllTrans

Aplikasi ini mengharuskan Anda untuk mendaftar untuk gratis kunci dari Microsoft atau Yandex terjemahan. Lihat petunjuk di bawah ini tentang cara untuk mendapatkan kunci.

Yandex Translate mendukung bahasa-bahasa lain, memiliki proses pendaftaran jauh lebih mudah, dan memerlukan tidak ada kartu kredit.

Microsoft Translate memberikan terjemahan yang lebih baik, tetapi membutuhkan kartu kredit untuk mendaftar. (Tapi jangan khawatir, tidak ada yang akan dikenakan biaya.)

  1. Pastikan AllTrans app diaktifkan dalam Xposed Installer -> modul.
  2. Meluncurkan AllTrans app. Di tab "Pengaturan Global", mengaktifkan "Menggunakan Yandex daripada Microsoft" jika Anda ingin menggunakan Yandex. Menonaktifkan menggunakan Microsoft.
  3. Pilih "Menerjemahkan dari bahasa" dan "Menerjemahkan ke bahasa".
  4. Di tab "Apps untuk menerjemahkan", menemukan aplikasi yang Anda inginkan untuk menerjemahkan, klik kotak centang di samping itu. Jika kotak centang tidak terlihat, cobalah memutar telepon ke samping ke dalam modus lansekap.
  5. Menutup dan restart aplikasi yang Anda inginkan diterjemahkan - itu harus diterjemahkan!

Bagaimana untuk mendapatkan kunci menerjemahkan Yandex

  1. Pergi ke Yandex menerjemahkan API Website
  2. Jika Anda belum memiliki akun, klik "daftar" dan mendaftar untuk account baru.
  3. Setelah mendaftar, menyetujui perjanjian pengguna dan klik "mendapatkan API Key (kunci lisensi kedaluwarsa".
  4. Anda telah berhasil menerima kunci Anda!

Tombol berlangganan adalah sesuatu seperti "trnsl.1.1.201701..."

Cara mendapatkan Microsoft Azure menerjemahkan kunci

  1. Mendaftar untuk sebuah account Microsoft Azure.
    Jika Anda belum memiliki akun Azure, sign up untuk account Microsoft Azure.
  2. Setelah Anda memiliki akun, masuk ke Azure Portal.
  3. Menambahkan "Microsoft Translator API berlangganan" ke akun Azure.
    1. Memilih berlangganan "bayar-saat". Akan memerlukan kartu kredit, tapi jangan khawatir, tidak akan dikenakan.
    2. Pilih opsi "+ baru".
    3. Pilih "Kecerdasan" dari daftar layanan.
    4. Pilih "Kognitif Layanan api".
    5. Pilih opsi "API jenis".
    6. Pilih "Teks terjemahan".
    7. Di bagian "Harga Tier", pilih "Gratis tingkat (F0)".
    8. Mengisi seluruh formulir, dan tekan tombol "Create".
  4. Mengambil Anda "otentikasi kunci".
    1. Pergi ke "Semua sumber daya" dan pilih Microsoft Translator API Anda berlangganan.
    2. Pergi ke opsi "Kunci" dan menyalin langganan kunci untuk mengakses layanan.

Tombol berlangganan adalah sesuatu seperti "321dcba..."

Pemecahan masalah

App tidak sedang diterjemahkan dalam setiap app

  • Jika aplikasi tidak sedang diterjemahkan, check adalah "AllTrans" diaktifkan di Xposed Installer -> modul.
  • Juga Periksa apakah Anda "Microsoft Yandex langganan kunci" benar.

Diterjemahkan app terjebak dalam layar pembuka

  • Klik pada nama app di "Apps untuk menerjemahkan" tab, dan klik pada "Klik untuk jelas terjemahan Cache".
  • Jika ini adalah pertama kali Anda mulai menerjemahkan aplikasi ini, klik pada nama app di tab "Apps untuk menerjemahkan", dan "Lain pengaturan" mengisi "Penundaan sebelum Replcaing dengan diterjemahkan teks"-2000.

Diterjemahkan app adalah "Kekuatan Close"

  • Klik pada nama app di "Apps untuk menerjemahkan" tab, dan klik pada "Klik untuk jelas terjemahan Cache".
  • Klik pada nama app di tab "Apps untuk menerjemahkan", dan di "pengaturan lain" mengisi "Penundaan sebelum Replcaing dengan diterjemahkan teks"-2000.
  • Klik pada nama app di "Apps untuk menerjemahkan" tab, dan dalam "Pengaturan lain" mengisi "Penundaan sebelum memulai untuk menerjemahkan WebViews"-2000.

Aplikasi tertentu yang tidak diterjemahkan

  • Klik pada nama app di "Apps untuk menerjemahkan" tab, dan dalam "Pengaturan lain" mengisi "Penundaan sebelum memulai untuk menerjemahkan WebViews"-2000.
  • Jika beberapa bagian dari aplikasi yang masih tidak diterjemahkan, klik pada nama app "Apps untuk menerjemahkan" tab, dan mengaktifkan "Agresif Mode".

Aplikasi permainan yang tidak diterjemahkan

  • Karena keterbatasan teknis, "AllTrans" tidak akan bekerja dengan banyak permainan. Tidak ada yang dapat dilakukan tentang hal ini.

Jika Anda masih memiliki masalah seperti "Kekuatan tutup" atau bagian dari sebuah aplikasi yang tidak diterjemahkan

Hubungi saya untuk dukungan. Bila Anda melakukannya, saya akan membutuhkan atleast informasi berikut-

  1. Nama app, bersama dengan link untuk men-download aplikasi dari beberapa toko APK seperti Google Play Store, CoolApk, dll.
  2. Telepon Android versi.
  3. "Logcat" Anda mencoba untuk membuka aplikasi yang memberikan Anda masalah. Anda dapat menggunakan aplikasi yang tersedia secara bebas di Play Store untuk ini, seperti yang satu ini

Setelah Anda memiliki hal di atas, hubungi saya di XDA mendukung benang.
